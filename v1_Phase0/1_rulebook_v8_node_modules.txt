# JavaScript + PixiJS v7→v8 Rulebook v3.1 - node_modules統合版

## 1. 基本理念（node_modules統合対応版）

### 1.1 核心方針
- **段階的品質向上**: v7完成→v8移行の安定的発展戦略
- **node_modules統合**: CDN依存からローカル依存への安定移行・CORS制限回避
- **非破壊的変形**: ベクターベース変形によるアニメ制作品質確保
- **エコシステム活用**: 車輪の再発明回避・@pixiエコシステム完全活用
- **AI協働開発**: 必要ファイルのみ添付での効率的開発・保守
- **統合システム設計**: 座標変換・履歴管理・デバッグの統一化推進

### 1.2 実装原則（node_modules対応版）
- **SOLID・DRY原則徹底**: 単一責任・責務分離・重複コード排除
- **500行ファイル制限**: 超過時は機能別分割・依存関係最小化
- **node_modules統合**: ローカルライブラリ読み込み・オフライン動作対応
- **JavaScript Pure**: ESM/TypeScript混在禁止・Pure JavaScript維持
- **非破壊品質保証**: アニメ制作における品質劣化完全防止

### 1.3 Phase移行戦略
- **Phase1**: HTML単体→node_modules分割移行（v7基盤）
- **Phase2**: 描画機能完成・レイヤーシステム（v7完成版）  
- **Phase3**: アニメーション・GIF機能（v7最終版）
- **Phase4**: PixiJS v8移行・120FPS・SPINE対応

### 2.3 pixi-extensions.js統合システム
```javascript
// libs/pixi-extensions.js - PixiJS拡張統合基盤
class PixiExtensionsManager {
    constructor() {
        this.extensions = new Map();
        this.initialized = false;
    }
    
    // ライブラリ検出・統合
    async initialize() {
        console.log('🔧 PixiJS拡張ライブラリ統合開始...');
        
        // 基本ライブラリ検証
        this.validateCoreLibraries();
        
        // 拡張ライブラリ統合
        this.integrateUILibrary();
        this.integrateLayersLibrary();
        this.integrateGIFLibrary();
        this.integrateUtilityLibraries();
        
        this.initialized = true;
        console.log('✅ PixiJS拡張ライブラリ統合完了');
    }
    
    validateCoreLibraries() {
        if (!window.PIXI) throw new Error('PixiJS が読み込まれていません');
        console.log(`✅ PixiJS v${PIXI.VERSION} 検出`);
    }
    
    integrateUILibrary() {
        if (window.PIXI && window.PIXI.UI) {
            this.extensions.set('ui', {
                available: true,
                components: window.PIXI.UI
            });
            console.log('✅ @pixi/ui 統合完了');
        }
    }
    
    integrateLayersLibrary() {
        if (window.PIXI && window.PIXI.display) {
            this.extensions.set('layers', {
                available: true,
                Layer: window.PIXI.display.Layer,
                Group: window.PIXI.display.Group
            });
            console.log('✅ @pixi/layers 統合完了');
        }
    }
    
    integrateGIFLibrary() {
        if (window.PIXI && window.PIXI.AnimatedGIF) {
            this.extensions.set('gif', {
                available: true,
                AnimatedGIF: window.PIXI.AnimatedGIF
            });
            console.log('✅ @pixi/gif 統合完了');
        }
    }
    
    integrateUtilityLibraries() {
        // GSAP統合
        if (window.gsap) {
            this.extensions.set('gsap', {
                available: true,
                gsap: window.gsap
            });
            console.log('✅ GSAP 統合完了');
        }
        
        // Lodash統合
        if (window._) {
            this.extensions.set('lodash', {
                available: true,
                _: window._
            });
            console.log('✅ Lodash 統合完了');
        }
        
        // Hammer.js統合
        if (window.Hammer) {
            this.extensions.set('hammer', {
                available: true,
                Hammer: window.Hammer
            });
            console.log('✅ Hammer.js 統合完了');
        }
    }
    
    // 機能チェック
    hasFeature(feature) {
        const extension = this.extensions.get(feature);
        return extension && extension.available;
    }
    
    // コンポーネント取得
    getComponent(library, component) {
        const extension = this.extensions.get(library);
        return extension && extension.available ? extension[component] : null;
    }
    
    // 統計情報
    getStats() {
        return {
            total: this.extensions.size,
            available: Array.from(this.extensions.values()).filter(ext => ext.available).length,
            libraries: Array.from(this.extensions.keys())
        };
    }
}

// グローバル公開
window.PixiExtensions = new PixiExtensionsManager();
```

### 2.4 Pure JavaScript読み込みパターン
```javascript
// js/main.js - Pure JavaScript エントリーポイント
class AppLoader {
    async init() {
        // PixiJS拡張ライブラリ初期化
        await window.PixiExtensions.initialize();
        
        // アプリケーション初期化
        const appCore = new AppCore();
        await appCore.init();
        
        // 動的機能読み込み
        this.startFeatureLoading();
    }
    
    startFeatureLoading() {
        // 条件付き読み込み（Pure JavaScript）
        if (this.needsPenTool()) {
            this.loadScript('js/tools/pen-tool.js');
        }
        
        if (this.needsLayerSystem()) {
            this.loadScript('js/layers/layer-manager.js');
        }
    }
    
    loadScript(src) {
        return new Promise((resolve, reject) => {
            const script = document.createElement('script');
            script.src = src;
            script.onload = resolve;
            script.onerror = reject;
            document.head.appendChild(script);
        });
    }
}
```

class UIManager {

        // 拡張ライブラリ使用前チェック

        this.hasLayers = window.PixiExtensions.hasFeature('layers');

            // @pixi/ui使用

            return new Layer();
        } else {
            // 基本Container使用

        }
    }

// js/transform/vector-transform.js
class VectorTransform {
    applyTransform(layer, transform) {
        // ✅ ベクターデータに変形行列適用（非破壊）
        layer.transform.setFromMatrix(transform);
        
        // ✅ 履歴に変形行列記録（完全復元可能）
        MemoryManager.record('transform', {
            layerId: layer.id,
            transform: transform.clone(),

        });

    // ❌ 禁止: ラスター変換（品質劣化）

class PerformanceMonitor {
    static validateVectorQuality(graphics) {
        // ベクター品質検証
        if (graphics instanceof PIXI.Graphics) {
            return graphics.geometry.points.length > 0;

## 10. 禁止事項（強化版）

### 10.1 基本禁止事項
- **ESM/TypeScript混在**: Pure JavaScript維持・type="module"禁止
- **CDN依存**: node_modules使用可能ライブラリのCDN使用禁止
- **循環依存**: ファイル間循環依存の完全禁止
- **ラスター変換**: ベクターデータの破壊的変換禁止

### 10.2 node_modules統合固有禁止事項
- **CDN迂回**: node_modules利用可能ライブラリのCDN使用禁止
- **バージョン混在**: package.jsonと異なるバージョン使用禁止
- **未検証使用**: PixiExtensions検証を経ない直接使用禁止

### 10.3 Pure JavaScript固有禁止事項
- **ES Module使用**: import/export文・type="module"禁止
- **TypeScript混在**: .tsファイル・型定義混在禁止
- **非同期import**: dynamic import禁止・script要素使用
