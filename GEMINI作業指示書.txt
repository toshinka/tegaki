## 今回の前提

- 標準描画は PixiJS v8.17.0 + RenderTexture へのライブラスター焼き込み。
- レイヤーは `Container + Sprite(RenderTexture)`。
- `animationSystem.init()` は現行描画へ副作用が出るため使わない。
- WebGPU / SDF / MSDF / WebGL2 Mesh 経路は凍結継続。
- `dist/` は成果物として必要ない限り差分に残さない。
- 大規模リネーム、ファイル移動、未接続ファイル削除、EventBus payload 変更は実装しない。候補として記録するだけ。

### 4. 小UI棚卸し

オーナーメモ追加分を整理する。

確認対象：

- レイヤーサムネイルの削除ボタンが見えないがクリック領域だけある問題。
- popup の `X` ボタン有無とデザイン統一。
- popup 外クリックで閉じる仕様との共存。
- レイヤー左右フリック削除、ピンチ操作などタッチ系要望の実装候補。

低リスクなら直してよい：

- 消えている削除ボタンの CSS 表示復旧。
- 既存 class の軽微な見た目統一。

実装しない：

- フリック削除。
- ピンチ操作。
- popup の大規模再設計。

### 5. Phase 1i 残件確認

短く確認する：

- V確定後サムネイルが即時更新されるか。
- V反転が即表示されるか。
- Shiftドラッグが拡縮/回転で排他になっているか。
- History ステータスが破綻していないか。

狭い不具合なら直してよい。履歴/RenderTexture/座標に深く絡む場合は `PROGRESS.md` に書いて止める。
