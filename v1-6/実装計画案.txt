1. より高度な筆圧表現（太さ以外への適用）
詳細: 現在、筆圧は線の太さに適用されていると思いますが、これをさらに拡張できます。
方法:
透明度 (アルファ値): 筆圧が弱いほど線が薄くなるように、ctx.globalAlpha を筆圧に応じて調整します。
色相/彩度: 筆圧に応じて線の色をわずかに変化させる（例: 強く描くと色が濃くなる、または赤みがかる）といった表現も理論上は可能です。

2. スムージングアルゴリズムの改善 (より高度な曲線)
詳細: 現在 quadraticCurveTo を使って中間点で滑らかにしていますが、さらに複雑な曲線アルゴリズムを導入することで、より自然で美しい曲線を描くことができます。
方法:
B-スプライン曲線やCatmull-Romスプライン: これらのアルゴリズムは、複数の点を通過し、より「自然な」曲線を生成します。特にCatmull-Romスプラインは、与えられた点を「必ず通る」特性があり、直感的な描画に適しています。
線の入り抜き（ tapering ): 線を引く開始点と終了点で、線の太さを滑らかに細くしていく表現。筆の入りや抜きを再現し、よりプロフェッショナルな見た目になります。
メリット: 非常に滑らかで芸術的な曲線表現が可能になります。線の入り抜きは特に表現力を高めます。
デメリット: 実装が quadraticCurveTo よりも複雑になり、計算負荷も増える可能性があります。デバッグも難しくなるかもしれません。

✅ 検証1：ImageData導入と描き味アップの干渉の有無
🔍 結論：基本的には干渉しない。ただし注意点が2つ。
機能	ImageDataとの干渉	解説
筆圧 → 太さ	❌ なし	現在も ctx.lineWidth に反映してるため問題なし
筆圧 → 透明度 (globalAlpha)	⚠️ 若干注意	ctx.putImageData() では透明度がそのまま保存される。描画中のアルファ合成では globalAlpha が乗算される点に注意
筆圧 → 色変化（彩度など）	❌ なし	RGB値を事前に調整するので、ImageData側には影響なし
スムージング改善	❌ なし	ベクターデータ処理。描画API側の話であり、ImageDataとは独立
線の入り抜き	❌ なし	α・サイズ制御の応用であり、Pixelレベルの保存形式には干渉しない

📝 ただし注意点：
描画中の ctx.globalAlpha の扱いに注意
putImageData() はアルファ合成を無視してピクセル単位で上書きされるため、プレビューやキャンバス描画は drawImage() を併用する必要があります。

スムージングが複雑になれば履歴（ImageData保存）の容量や負荷も比例して増加
Undo/Redoの保存形式としてImageDataを多用する設計にしているなら、描画途中のキャプチャを「間引き」や「バウンダリポイントごとの保存」にすると良いです。

class OffscreenRenderer {
    constructor() {
        this.workerPool = new WorkerPool(4); // 4スレッド
        this.renderQueue = new PriorityQueue();
    }

    async renderLayer(layer) {
        const offscreen = new OffscreenCanvas(layer.width, layer.height);
        const renderTask = {
            layer: layer,
            canvas: offscreen,
            priority: layer.isVisible ? 1 : 0
        };

        this.renderQueue.add(renderTask);
        await this.processRenderQueue();
    }

    async processRenderQueue() {
        while (!this.renderQueue.isEmpty()) {
            const task = this.renderQueue.next();
            const worker = await this.workerPool.getWorker();
            
            worker.postMessage({
                type: 'render',
                layer: task.layer,
                canvas: task.canvas
            }, [task.canvas]);
        }
    }
}

            const [p1, p2] = pointers.values();
            
            // ピンチ判定
            const distance = this.getDistance(p1, p2);
            if (Math.abs(distance - this.lastDistance) > 10) {
                return {
                    type: 'pinch',
                    scale: distance / this.lastDistance
                };
            }

            // 回転判定
            const angle = this.getAngle(p1, p2);
            if (Math.abs(angle - this.lastAngle) > 0.1) {
                return {
                    type: 'rotate',
                    angle: angle - this.lastAngle
                };
            }
        }
    }
}

class StrokeStabilizer {
    constructor() {
        this.points = [];
        this.weight = 0.8; // 安定化の強さ（0-1）
    }

    addPoint(point) {
        this.points.push(point);
        if (this.points.length > 4) {
            this.points.shift();
        }
        return this.getStabilizedPoint();
    }

    getStabilizedPoint() {
        if (this.points.length < 2) return this.points[0];

        const last = this.points[this.points.length - 1];
        const stabilized = {
            x: 0,
            y: 0,
            pressure: 0
        };

        let totalWeight = 0;
        this.points.forEach((p, i) => {
            const weight = Math.pow(this.weight, this.points.length - 1 - i);
            stabilized.x += p.x * weight;
            stabilized.y += p.y * weight;
            stabilized.pressure += p.pressure * weight;
            totalWeight += weight;
        });

        return {
            x: stabilized.x / totalWeight,
            y: stabilized.y / totalWeight,
            pressure: stabilized.pressure / totalWeight
        };
    }
}

class LayerCompositor {
    constructor() {
        this.compositingCanvas = document.createElement('canvas');
        this.ctx = this.compositingCanvas.getContext('2d');
    }

    compositeVisibleLayers(layers, dirtyRect) {
        // クリッピング領域の設定
        this.ctx.save();
        this.ctx.beginPath();
        this.ctx.rect(dirtyRect.x, dirtyRect.y, dirtyRect.width, dirtyRect.height);
        this.ctx.clip();

        // 可視レイヤーのみを合成
        layers
            .filter(layer => layer.visible)
            .forEach(layer => {
                this.ctx.globalAlpha = layer.opacity;
                this.ctx.globalCompositeOperation = layer.blendMode;
                this.ctx.drawImage(
                    layer.canvas,
                    dirtyRect.x, dirtyRect.y,
                    dirtyRect.width, dirtyRect.height,
                    dirtyRect.x, dirtyRect.y,
                    dirtyRect.width, dirtyRect.height
                );
            });

        this.ctx.restore();
    }
}
