1.2 描画処理の統一
CanvasManager.drawPixel(layer, x, y, color, size):
与えられた layer.imageData の指定された (x, y) 座標に、color と size (ブラシサイズ) に応じたピクセルを描画する処理を実装。
ブラシサイズに応じて、単一ピクセルだけでなく、円形（または四角形）の領域を塗りつぶすように実装すること。
このメソッド内で dirtyRect（変更された最小の矩形領域）を更新すること。
CanvasManager.drawLine(layer, x1, y1, x2, y2, color, size):
Bresenham's line algorithm などを使用して、layer.imageData の (x1, y1) から (x2, y2) まで線を引く処理を実装。
線の太さは size で指定されたブラシサイズを適用すること。
このメソッド内で dirtyRect を更新すること。
CanvasManager.draw(activeLayer, currentX, currentY, prevX, prevY, color, size) の変更点:
このメソッド内で、上記の drawPixel や drawLine を呼び出して activeLayer.imageData を直接操作するように変更する。
displayCtx.putImageData を使って、変更された領域のみを再描画する。

JavaScript

// CanvasManager クラス内
renderDirtyRect() {
    const rect = this.dirtyRect;

    if (rect.minX === Infinity) return; // 変更なし

    // 少し余裕を持たせる
    const margin = 2; // ブラシのアンチエイリアスなども考慮
    const x = Math.max(0, rect.minX - margin);
    const y = Math.max(0, rect.minY - margin);
    const width = Math.min(this.displayCanvas.width - x, rect.maxX - rect.minX + margin * 2);
    const height = Math.min(this.displayCanvas.height - y, rect.maxY - rect.minY + margin * 2);

    // 該当領域のみ再描画
    this.displayCtx.clearRect(x, y, width, height); // 既存の描画をクリア

    this.app.layerManager.layers.forEach(layer => {
        if (layer.visible) {
            // ImageData全体を一度 displayCtx に描き、そのうち dirtyRect の部分だけをコピーする
            // putImageData(imageData, dx, dy, dirtyX, dirtyY, dirtyWidth, dirtyHeight)
            this.displayCtx.putImageData(
                layer.imageData, 
                0, 0, // imageDataの (0,0) を displayCtx の (0,0) に描画
                x, y, width, height // そのうち、imageDataの (x,y) から width x height の領域をコピー
            );
        }
    });
    this.resetDirtyRect(); // 描画後、dirtyRectをリセット
}
