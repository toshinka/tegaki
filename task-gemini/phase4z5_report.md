# Phase 4z5 — Initial ClipAsset Auto-Seed MVP (GEMINI報告)

## 1. 実施内容

### 🌱 初期クリップの自動生成 (Auto-Seed)
- **`ui/animation-table-popup.js`**
    - **`_ensureInitialClipAssetSeed()` の実装**: アニメーションテーブルを最初に開いた際、現在のキャンバスの描画内容を自動的に1コマ目（Frame 1）のクリップとして登録する機能を実装しました。
    - **ターゲット選定ロジック**:
        1. 現在アクティブなレイヤーに対応するレーンを優先。
        2. アクティブレイヤーがない場合は、一番上の通常レーン（Folder以外）を選択。
        3. 背景レイヤー（Background）は自動シードの対象から除外。
    - **重複防止**: すでに1コマ目にクリップが存在する場合は実行をスキップします。
    - **セッション管理**: `initialClipAssetSeeded` フラグにより、アプリ起動中一度だけ実行されるように制御しました。

### 🎨 UX の向上
- **「空のタイムライン」の解消**: これまでタイムラインを開いただけでは何も表示されず、手動でスロットをクリックする必要がありましたが、今回の改修により「今描いている絵」が即座にアニメーションの1コマ目として認識されるようになりました。
- **自動選択**: 生成されたクリップは自動的に選択状態になり、すぐに DURATION 調整や EDIT モードへの移行が可能です。
