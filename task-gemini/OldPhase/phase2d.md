## UI 方針

透明度ポップアップ案は `画像資料/透明度スライダーポップアップ案.png` を参考にする。

候補構成:

1. 透明度プリセット
   - 0 / 25 / 50 / 75 / 100 などの小ボタン。
2. 透明度スライダー
   - 既存の `setLayerOpacity()` / `layer:opacity-changed` 経路を再利用する。
3. 合成モード
   - 通常 / 乗算 / 加算 / オーバーレイなど。
   - 初回は UI 表示だけ、または既存フィールド確認に留めてよい。
4. クリッピング切替
   - ON の時、レイヤー行のクリップ状態欄にアイコンを出す。
   - 本体処理は後続。保存/履歴/描画への影響があるため、Gemini は実装しない。
