## 各タスクの詳細仕様・実装ステップ

### Task 1: お手本動画（MP4/GIF）の読み込みとフレーム分解
*   **内容**: 既存の MP4 動画や GIF 画像を読み込み、最初の24フレーム等をお手本（テンプレート）としてタイムラインにロードする機能。
*   **仕様**:
    *   ユーザーがファイルをドラッグ＆ドロップまたはファイル選択ポップアップから指定。
    *   ローカルの `ffmpeg` もしくは WebCodecs API を使ってフレーム画像へと分解。
    *   分解された画像を「AI参考（ai_reference）レーン」へお手本アセットとして自動流し込みする。

### Task 2: AI Adapter（ローカルAPIサーバー中継）の構築
*   **内容**: CORS制約や LLM での日本語指示文の英語化、ffmpeg による動画分解などを安全に行うため、Node.js などのローカルAPI中継サーバーを設置。

### Task 3: タイムライン上への「始点・終点アセット」指定UI
*   **内容**: セル右クリック等から「このフレームを始点に設定」「終点に設定」を呼び出す。タイムライン上に `[S]` / `[E]` 等のマークを表示。

### Task 4: 日本語プロンプト指示入力とLLM英語変換
*   **内容**: 日本語の動作説明文（「髪が揺れる」等）を入力し、ローカルサーバー経由で LLM に投げ、英語プロンプト・モーション指示へと自動翻訳させる。

### Task 5: ComfyUI / WAN2.2 連携による動画生成
*   **内容**: API経由で ComfyUI に接続し、テンプレート化された WAN2.2 ワークフローに画像と英語プロンプトを差し込んで短尺動画を生成。

### Task 6: 生成連番画像の「AI参考レーン」自動配置とオニオンスキン
*   **内容**: 生成結果の連番画像を不透明度 30%〜40% の編集ロックされた「AI参考レーン」に展開。その上からユーザーが新規レイヤーで清書（トレース）を行えるようにする。
