## 各タスクの詳細仕様・実装ステップ

### Task 1: 非破壊変形（劣化しない変形）の導入
*   **内容**: ラスター画像の移動・回転・拡縮を繰り返すことによる「ピクセル補間劣化」を防ぐ仕組み。
*   **仕様**:
    *   **変形中 (VキーON時)**: レイヤーの元の RenderTexture を一時的に `OriginalTexture` として別バッファに退避。操作中は PixiJS Sprite の `transform`（`position/scale/rotation`）値だけを直接変更してGPUプレビュー表示（劣化なし）。
    *   **変形確定 (VキーOFF時)**: 退避しておいた `OriginalTexture` から、最終的な累積トランスフォーム行列を適用して1回だけ新規 RenderTexture へ焼き込む。
    *   **XY移動の劣化ゼロ化**: 平行移動のみの場合、確定時に座標値を四捨五入して整数（`Math.round`）に吸着。ピクセル補間を一切発生させず、完全に劣化なしの移動を実現。

### Task 2: 逆クリッピング（反転クリップ）の実装
*   **内容**: 下のレイヤーの描画部分を透過・マスク反転させて上のレイヤーを表示させる機能。
*   **仕様**:
    *   通常クリッピング（下のレイヤーの描画領域にのみ表示）に加え、下のレイヤーの「透明な領域」にのみ上のレイヤーの描画を表示する「逆クリッピング」をサポート。
    *   **操作**: レイヤーパネルのクリップアイコンをクリックするたびに、`クリップなし ➡️ 通常クリップ ➡️ 逆クリップ` の順でトグル（循環）切り替え。
    *   **UI表示**: 逆クリップ状態のときは、アイコンの配色を反転させたり、専用の反転マスク用アイコン（例: 斜線入りクリップや禁止マークのオーバーレイ）に動的に変化させ、一目で区別できるようにする。
    *   **PixiJS実装**: レイヤー描画合成時、ブレンドモード（`destination-out` や `source-out`）のブレンド組み合わせ、もしくはステンシルマスクの反転設定を適用する。

### Task 3: クリップボード画像ペースト＆自動リサイズ
*   **内容**: 外部（Webブラウザやエクスプローラー等）でコピーされた画像を `Ctrl+V` でアクティブレイヤーに直接ペースト。アスペクト比を維持したまま、キャンバスの最大サイズに収まるように自動縮小リサイズして配置。

### Task 4: アクティブCAFのPSD出力
*   **内容**: 現在選択されている「アクティブなCAF（アニメーション用フォルダ・コンテナ）」の内部にある画像レイヤー階層のみをPSDファイルとして保存する。タイムテーブル情報等は除外し、外部ツール連携の互換性を確保。

### Task 5: PSDインポート＆自動リサイズ
*   **内容**: PSDファイルのインポート。読み込んだレイヤー構造を展開し、キャンバスサイズを超える大きな画像アセットはアスペクト比を維持して自動的に縮小リサイズして適合させる。

### Task 6: 手ブレ補正（LazyBrush）の本格実装
*   **内容**: 設定スライダー（`smoothing` 0.0〜1.0）を描画側（PointerHandler等）へ動的に反映する手ブレ補正。

### Task 7: キャンバスサイズ制限緩和（2500px）
*   **内容**: 上限を `2500px`へ引き上げ。リサイズ時の不要メモリ解放を徹底。

### Task 8: 直線・円・矩形など図形描画ツール
*   **内容**: ドラッグでプレビュー、離して確定する図形ツール。`Shift` 押下で45度・正円・正方形スナップ。

### Task 9: スクリーントーン（網点・グラデ・ストライプ）
*   **内容**: 網点やストライプをブラシテクスチャ、あるいは選択範囲へのパターン流し込みとして実装。

### Task 10: 集中線・ウニフラッシュ＆定規ツール
*   **内容**: 中心点を基準に線が引き寄せられる集中線ツール、または回転に対応した仮想定規ツール。

### Task 11: 漫画のコマ用自動マスキング
*   **内容**: コマ枠の形状を定義し、その外側の描画を不可にするマスク。枠の変形にマスクが追従する。

### Task 12: スタンプ描画エンジンへの置換（将来到達点・保留）
*   **内容**: perfect-freehand方式の重なりムラ（三角分割ノイズ）を解消するため、等間隔ブラシスタンプ ＋ スクラッチバッファ蓄積方式へ移行する。
