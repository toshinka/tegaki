## Phase 4 用 ライブラリ選定早見表（2025年7月更新）

### ✅ 採用済みライブラリ

| ライブラリ名     |    用途             | コメント                            |
| ---------------- | ------------------- | ----------------------------------- |
| twgl.js          | WebGL描画簡略化     | WebGLのFBO/シェーダ記述を簡単にする |
| gl-matrix        | 行列演算            | レイヤー変形などで必須              |
| Dexie.js         | IndexedDB保存       | 安定・非同期・高速                  |
| perfect-freehand | ペン補間（導入中）  | Paper.jsから移行                    |

---

### 🕒 採用予定ライブラリ

| ライブラリ名       | 用途                   | 採用予定フェーズ / 理由               |
| ------------------ | ---------------------- | ------------------------------------- |
| Tweakpane          | UI調整                 | Phase 4B UI刷新                       |
| Pointer Events API | 筆圧/チルト対応        | Phase 4B or D後半で導入               |
| GSAP               | アニメーション制御     | Phase 4Cタイムライン                  |
| Konva.js           | タイムラインUI構築     | Phase 4Cで検討                        |
| squoosh            | 書き出し時の画像圧縮   | Phase 4E                              |
| zod                | データスキーマバリデーション | 書き出し/共有構造で使用予定     |

---

### ❌ 不採用/保留ライブラリ（理由付き）

| ライブラリ名 | 用途               | 理由                                  |
| ------------ | ------------------ | ------------------------------------- |
| Paper.js     | パス補間・ベジェ   | 破線描画問題、補間用途に不適          |
| Fabric.js    | フルCanvas管理     | 自前Canvas制御が重くWebGL統合が困難   |
| Rough.js     | 手描き風           | 独自描画スタイルが方向性に合わない    |
| Lucide React | アイコン/UI        | React専用。非React構成では非適合      |
| Radix UI     | アクセシブルUI     | React前提。構造上導入不可             |
| Lottie Web   | AEアニメ再生       | 将来的に導入検討（Phase 5X以降）      |
| Three.js     | 3D/WebGL           | AE風エフェクト・WebGPU用に将来検      |
| Babylon.js   | 3D/物理・ゲーム系  | WebGL層が複雑。Phase 5以降限定候補    |
