### **レイヤー・フォルダ同期システム**
```
┌─────────────────────────────────────┐
│       アニメモード設定              │  ← 設定パネル
├─────────────────────────────────────┤
│ □ 全カット同期モード                │  ← チェックボックス
│   レイヤー・フォルダの作成・削除・   │     説明テキスト
│   移動を全カットに反映します        │
├─────────────────────────────────────┤
│ 同期対象:                          │
│ ☑️ レイヤー作成・削除               │  ← 個別チェック
│ ☑️ フォルダ作成・削除               │
│ ☑️ 階層移動                        │
│ ☑️ 名前変更                        │
└─────────────────────────────────────┘
```
※考え方としてはカットがタイムラインの時間の縦軸でフォルダやレイヤーはトラックの横軸。どのカットも基本的には同じ名前と構造のトラック＝レイヤーとフォルダを保持してる。
※同期システムは設定パネルの中に設ける。

④## ⌨️ ショートカットキー仕様

### 基本操作
- **アンドゥ/リドゥ**: `Ctrl + Z` / `Ctrl + Y`
- **レイヤー内消去**: `DEL`
- **ビューリセット**: `Ctrl + 0`

### キャンバス操作
- **移動**: `Space + ドラッグ` | `方向キー`
- **拡縮・回転**: `Shift + Space + ドラッグ` | `Shift + 方向キー`
- **反転**: `H`（左右） | `Shift + H`（上下）

### レイヤー操作
- **レイヤー移動**: `↑` / `↓`
- **フォルダ移動**: `Shift + ↑` / `Shift + ↓`

### レイヤー内絵画操作
- **移動**: `V + ドラッグ`
- **拡縮・回転**: `V + Shift + Space + ドラッグ`
- **反転**: `V + H` | `V + Shift + H`

### ツール切り替え
- **ペン**: `P` | **消しゴム**: `E` | **スポイト**: `I`
- **塗りつぶし**: `G` | **範囲選択**: `M` | **テキスト**: `T`

### ツール設定
- **ペンサイズ**: `P + [` / `P + ]`
- **消しゴムサイズ**: `E + [` / `E + ]`

### UI操作
- **ポップアップ/レイヤーパネル表示のON/OFF切り替え**: `TAB`
※TABキーででON/OFFするパネルは設定画面で決めれる。

### アニメーション専用

- **カット移動**: `←` / `→`
- **5カット飛ばし**: `Shift + ←` / `Shift + →`
- **再生/停止**: `ALT + Space`
- **シャトル**: `ALT + J` / `ALT + L`
- **フレーム移動**: `ALT + ←` / `ALT + →`
- **オニオンスキン切り替え**: `ALT + O`
