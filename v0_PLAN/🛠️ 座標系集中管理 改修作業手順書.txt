### Step 1-2: 重複座標処理の統一化
**作業方針**: 全ての座標変換を`CoordinateManager`経由に統一

**変更パターン例**:
```javascript
// 変更前（他ファイルでの直接座標変換）:
const rect = canvas.getBoundingClientRect();
const x = event.clientX - rect.left;
const y = event.clientY - rect.top;

// 変更後（CoordinateManager経由）:
const coords = this.coordinateManager.extractPointerCoordinates(
    event, 
    canvas.getBoundingClientRect()
);
const x = coords.canvas.x;
const y = coords.canvas.y;
```

**参考メソッド・シンボル**:
- `CoordinateManager.extractPointerCoordinates()`
- `CoordinateManager.screenToCanvas()`
