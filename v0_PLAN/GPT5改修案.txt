1. 現象

ペン描画・消しゴムは正常

Stroke 登録や描画フローは正常に動作している。

キャンバス移動（カメラ移動）が正しくない

Camera 座標だけが動き、Pixi stage は動かず描画内容は固定。

初動時に左方向にオフセット（約150pxくらい）が発生。

レイヤー管理が不正

LayerToolUI でレイヤー+ は動作する。

ただし アクティブレイヤー切替時に、以前のレイヤーで書いた Stroke が消せる。

Layer 層の概念が破綻しており、LayerID の管理や Stroke の紐付けが正しく反映されていない。

2. 原因分析
2.1 キャンバス移動の挙動

EnginePosition.setCamera(x,y) は camera 値のみ更新しており、Pixi stage には反映されていない。

現状の drawStroke() は World→Screen 変換で Camera を減算して描画しているが、既存の Stroke には適用されない。

→ そのため、Camera が動いても「見た目は動かない」状態。

初動オフセットは stage 初期位置 + Camera 初期値が不揃いで発生。

2.2 レイヤー概念の破綻

LayerToolUI は LayerID を管理しているが、Stroke がどの Layer に属するかの紐付けが不十分。

Stroke 登録時に stroke.layerId を正しく設定していない場合、Layer切替で全 Stroke が操作可能になってしまう。

また、Layer が描画順やアクティブ状態で Pixi Graphics をまとめて管理していないため、Layer の描画分離が失われている。

3. 改修案
3.1 Camera 反映の統合

Pixi stage を Camera に従って移動

function setCamera(x, y) {
  camera.x = x;
  camera.y = y;
  app.stage.x = -camera.x; // stage そのものを移動
  app.stage.y = -camera.y;
  MainController.safeNotify('camera-moved', { camera: {...camera} });
}


既存 Stroke の再描画

Camera 移動後、既存の Graphics をクリアし、再描画する

または 各 Stroke に layerId を保持し、Layer ごとに Pixi Container を分ける

3.2 Layer 分離の強化

Layerごとに Pixi Container を作成

const layerContainers = {}; // layerId -> PIXI.Container
function createLayer(name="layer") {
    const id = `layer-${layers.length+1}`;
    const layer = { id, name, points: [] };
    layers.push(layer);
    activeLayerId = id;
    const container = new PIXI.Container();
    app.stage.addChild(container);
    layerContainers[id] = container;
    MainController.safeNotify('layer-created', { layerId: id });
}


Stroke 描画時にアクティブレイヤー Container に描画

function drawStroke(stroke) {
    strokes.push(stroke);
    const pts = stroke.points.map(p=>({...p, screenX:p.x-camera.x, screenY:p.y-camera.y}));
    const g = new PIXI.Graphics();
    g.lineStyle(2, 0x000000);
    if(pts.length>0) g.moveTo(pts[0].screenX, pts[0].screenY);
    for(let i=1;i<pts.length;i++) g.lineTo(pts[i].screenX, pts[i].screenY);
    layerContainers[stroke.layerId].addChild(g);
    MainController.safeNotify('stroke-drawn', { layerId: stroke.layerId, points: pts });
}


アクティブ Layer 以外の Stroke は消しゴム操作不可

消しゴム操作は layerId === activeLayerId の場合のみ許可

Layerごとに Container を分けておけば、描画・消去の分離が容易

3.3 初期オフセット修正

app.stage の初期位置を (0,0) に揃える

Camera 初期値を (0,0) に固定

起動時にオフセットが加わらないように setCamera(0,0) を呼ぶ

4. 推奨フロー（改修後）
[UI操作]
   → LayerToolUI (レイヤーID付Stroke生成)
       → MainController.safeNotify
           → EnginePosition.drawStroke
               → layerContainers[activeLayerId] に描画
               → Camera考慮したWorld→Screen変換
Camera移動 → EnginePosition.setCamera
   → stage.x/y 更新 + 既存 Stroke 再描画 or Container移動


💡 ポイント

Pixi Container を Layer ごとに分離することでレイヤー概念を復活

Camera 移動時は stage そのものを移動 + 必要に応じて再描画

Stroke/Layer payload は循環参照なしで MainController に通知

消しゴムや描画フローが アクティブ Layer に限定されるようになる

   * @depends  engine-position.js, layer-tool-ui.js, error-service.js

  const MainController = (() => {

    function registerSatellite(name, module) {
      satellites[name] = module;
    }

    function safeNotify(type, payload) {
      let safePayload;
      try {
        JSON.stringify(payload);
        safePayload = payload;
      } catch(e) {
        console.warn("[MainController] Payload contains circular references:", type);
        safePayload = JSON.parse(JSON.stringify(payload, (key, value) => {
          if (key === 'parent' || key === 'children') return undefined;
          return value;
        }));
      }
      for (const key in satellites) {
        if (satellites[key].onEvent) satellites[key].onEvent({ type, payload: safePayload });
      }
    }

        await satellites.errorService.init();
        await satellites.enginePosition.init(document.getElementById('mainCanvas'));

        console.log("[MainController] Application started.");
      } catch (err) {
        satellites.errorService.reportError('APP_START_FAILED', err);
      }
    }

    return { registerSatellite, safeNotify, startApplication };
  })();

  MainController.registerSatellite('enginePosition', EnginePosition);
  MainController.registerSatellite('layerToolUI', LayerToolUI);

2. 衛星1：engine-position.js

/**
 * @module   EnginePosition
 * @role     描画エンジン + 座標管理
 * @depends  MainController
 * @provides init(canvas), drawStroke(stroke), setCamera(x,y), worldToScreen(), screenToWorld()
 * @notes    Camera/World/Screen座標系を統合、Pixi ContainerでLayer分離
 */

const EnginePosition = (() => {
  let app, camera = { x:0, y:0 }, strokes = [];
  const layerContainers = {}; // layerId -> PIXI.Container

  async function init(canvas) {
    app = new PIXI.Application({ view: canvas, width: canvas.width, height: canvas.height });
    console.log("[EnginePosition] Initialized.");
  }

  function registerLayerContainer(layerId) {
    if(!layerContainers[layerId]) {
      const container = new PIXI.Container();
      app.stage.addChild(container);
      layerContainers[layerId] = container;
    }
  }

  function drawStroke(stroke) {
    strokes.push(stroke);
    registerLayerContainer(stroke.layerId);

    const pts = stroke.points.map(p=>({...p, screenX:p.x-camera.x, screenY:p.y-camera.y}));
    const g = new PIXI.Graphics();
    g.lineStyle(2, 0x000000);
    if(pts.length>0) g.moveTo(pts[0].screenX, pts[0].screenY);
    for(let i=1;i<pts.length;i++) g.lineTo(pts[i].screenX, pts[i].screenY);

    layerContainers[stroke.layerId].addChild(g);

    MainController.safeNotify('stroke-drawn', { layerId: stroke.layerId, points: pts });
  }

  function setCamera(x, y) {
    camera.x = x; camera.y = y;
    app.stage.x = -camera.x;
    app.stage.y = -camera.y;
    MainController.safeNotify('camera-moved', { camera:{...camera} });
  }

  function worldToScreen(p) { return { x:p.x-camera.x, y:p.y-camera.y }; }
  function screenToWorld(p) { return { x:p.x+camera.x, y:p.y+camera.y }; }

  function onEvent(event) {
    // 例: MainController通知受信
  }

  return { init, drawStroke, setCamera, worldToScreen, screenToWorld, onEvent };
})();


3. 衛星2：layer-tool-ui.js

/**
 * @module   LayerToolUI
 * @role     レイヤー管理 + ツール + UI
 * @depends  MainController
 * @provides init(), createLayer(name?), deleteLayer(id), setActiveLayer(id), onUIButtonClick(event)
 * @notes    LayerID 安定生成、アクティブレイヤー制御、循環参照禁止
 */

const LayerToolUI = (() => {
  let layers = [], activeLayerId = null;
  const strokes = [];

  async function init() {
    console.log("[LayerToolUI] Initialized.");
  }

  function createLayer(name="layer") {
    const id = `layer-${layers.length+1}`;
    const layer = { id, name, points: [] };
    layers.push(layer);
    activeLayerId = id;
    MainController.safeNotify('layer-created', { layerId: id });
  }

  function setActiveLayer(id) {
    if(layers.find(l=>l.id===id)) activeLayerId = id;
    MainController.safeNotify('layer-activated', { id });
  }

  function drawStroke(points) {
    if(!activeLayerId) return;
    const stroke = { layerId: activeLayerId, points };
    strokes.push(stroke);
    MainController.safeNotify('path-ready-for-layer', { layerId: activeLayerId, points });
  }

  function eraseStroke(stroke) {
    if(stroke.layerId!==activeLayerId) return; // 他レイヤーは消せない
    // 消去処理
  }

  function onUIButtonClick(event) {
    if(event.type==="add-layer") createLayer();
  }

  function onEvent(event) {
    // 例: MainController通知受信
  }

  return { init, createLayer, setActiveLayer, drawStroke, eraseStroke, onUIButtonClick, onEvent };
})();

4. 衛星3：error-service.js

/**
 * @module   ErrorService
 * @role     エラー検知・通知
 * @depends  MainController
 * @provides reportError(code, details), showErrorNotification(msg)
 */

const ErrorService = (() => {
  let history = [];

  async function init() {
    console.log("[ErrorService] Initialized.");
  }

  function reportError(code, details) {
    history.push({ code, details, timestamp: Date.now() });
    showErrorNotification(`[${code}] ${details}`);
  }

  function showErrorNotification(msg) {
    console.error("[ErrorService] ", msg);
    MainController.safeNotify('error-occurred', { message: msg });
  }

  function onEvent(event) {
    // 他衛星からのエラー受信も処理可能
  }

  return { init, reportError, showErrorNotification, onEvent };
})();


✅ 修正ポイント

Layerごとに Pixi Container 分離 → 他レイヤーの Stroke を操作不可に。

Camera 移動時に Pixi stage 反映 → 描画内容も含めてスクロール。

Layer ID 安定化 → layer-${index+1}。

Stroke payload は循環参照なしで MainController に通知。

アクティブレイヤー判定で消しゴム制御。
