✅ 原因

PixiJS の Graphics や Sprite を普通に使うと、常に「上書き描画」 になります。
キャンバス2D API のような globalCompositeOperation = "destination-out" に相当する仕組みを入れないと「透明で消す」動作にはなりません。

現在の DrawingEngine ではたぶん「線を描く」と「消す」で同じ描画ルーチンを使っていて、色だけ変えている状態です。そのため「白で塗りつぶす＝消すように見えるけど実際は透明にならない」動きになっています。

✅ 対策

PixiJS v8 で消しゴムを実現するには ブレンドモード（合成モード） を切り替える必要があります。

ペン：通常描画

graphics.blendMode = PIXI.BLEND_MODES.NORMAL;
graphics.lineStyle(brushSize, brushColor, 1.0);


消しゴム：透明で塗り潰し

graphics.blendMode = PIXI.BLEND_MODES.ERASE;
graphics.lineStyle(brushSize, 0xFFFFFF, 1.0); // 色は任意、アルファは無視される


BLEND_MODES.ERASE は WebGL2 / PixiJS v5 以降 で追加された「透明で削る」モードです。
これを stroke / fill 前に指定することで「消しゴム」が実現できます。

✅ 修正例（drawing-engine.js の一部）
class DrawingEngine {
    constructor(main){
        this.main = main;
        this.currentTool = 'brush';
        this.graphics = new PIXI.Graphics();
        this.main.layers.stage.addChild(this.graphics);
    }

    setTool(tool){
        this.currentTool = tool;
    }

    startStroke(point){
        this.graphics.moveTo(point.x, point.y);
    }

    addPoint(point){
        this.graphics.lineTo(point.x, point.y);
    }

    endStroke(){
        this.main.notify({type:'tools.brushEnd'});
    }

    confirmStroke(){
        // 選択中のツールでブレンドモードを切り替える
        if(this.currentTool === 'eraser'){
            this.graphics.blendMode = PIXI.BLEND_MODES.ERASE;
            this.graphics.lineStyle(10, 0xFFFFFF, 1.0); // size=10の消しゴム
        } else {
            this.graphics.blendMode = PIXI.BLEND_MODES.NORMAL;
            this.graphics.lineStyle(2, 0x000000, 1.0); // size=2の黒ペン
        }
        this.graphics.strokePath();
    }
}
