問題点と原因解析
1. PixiJS アプリの初期化が行われていない

<div id="drawing-canvas"></div> を用意していますが、Pixi の Application を生成して drawing-canvas に app.canvas を追加していません。

そのため 実際の <canvas> 要素が DOM に存在せず、描画が一切行われない状態になっています。

2. レイヤーパネル初期化処理が不足

#layer-list に HTML の箱はありますが、JavaScript でレイヤー項目 (.layer-item) を生成する処理がありません。

そのため レイヤーパネルが常に空のままです。

3. ショートカットの定義はあるが、UI・描画と紐付いていない

CanvasTransformSystem などの仕組みは書かれていますが、PixiJS の stage に紐づけていないため、イベントを受けても視覚変化がありません。

改修案（コード追加例）

以下のコードを <script> 内の末尾（CanvasTransformSystem などのクラス定義の後）に追加してください。

// === PixiJS Application 初期化 ===
(async function initCanvas() {
    const app = new PIXI.Application({
        width: ENGINE_CONFIG.canvas.width,
        height: ENGINE_CONFIG.canvas.height,
        backgroundColor: 0xffffff,
        antialias: true,
        resolution: 1,
    });
    await app.init();

    // DOM に追加
    document.getElementById("drawing-canvas").appendChild(app.canvas);

    // メインのコンテナ
    const canvasContainer = new PIXI.Container();
    app.stage.addChild(canvasContainer);

    // 仮のレイヤー
    const baseLayer = new PIXI.Container();
    canvasContainer.addChild(baseLayer);

    // テスト描画
    const g = new PIXI.Graphics().rect(50, 50, 100, 100).fill(0xff0000);
    baseLayer.addChild(g);

    // 状態表示更新（FPS）
    app.ticker.add(() => {
        document.getElementById("fps").innerText = Math.round(app.ticker.FPS);
    });

    // レイヤーパネルに初期レイヤーを追加
    addLayerItem("背景", true);
})();

// === レイヤーパネル操作 ===
function addLayerItem(name, active = false) {
    const list = document.getElementById("layer-list");
    const item = document.createElement("div");
    item.className = "layer-item" + (active ? " active" : "");
    item.innerHTML = `
        <div class="layer-visibility"><svg viewBox="0 0 24 24"><circle cx="12" cy="12" r="6" stroke-width="2" fill="none"/></svg></div>
        <div class="layer-opacity">100%</div>
        <div class="layer-name">${name}</div>
        <div class="layer-thumbnail"><div class="layer-thumbnail-placeholder"></div></div>
        <div class="layer-delete-button"><svg viewBox="0 0 24 24"><line x1="18" y1="6" x2="6" y2="18"/><line x1="6" y1="6" x2="18" y2="18"/></svg></div>
    `;
    list.appendChild(item);
}

改修効果

drawing-canvas に Pixi の <canvas> が追加され、赤い四角が表示される → キャンバス表示確認可能。

layer-list に「背景」レイヤーが追加される → レイヤーパネルが見える。

今後 addLayerItem() をショートカットや #add-layer-btn と連動させれば、追加レイヤーが動的に増える。
