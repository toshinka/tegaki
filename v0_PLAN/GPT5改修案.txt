✅ 改修策

次の流れで修正すればOKです。

レイヤーパネルDOMを生成

<div id="layer-panel" class="layer-panel hidden">
  <div class="layer" data-id="0">背景</div>
  <div class="layer" data-id="1">レイヤー1</div>
</div>


初期状態は hidden クラスで非表示。

CSSで右サイドにパネルが出るようにする。

サイドバーのレイヤーアイコンにイベントを繋げる

document.getElementById("layerButton").addEventListener("click", () => {
  document.getElementById("layer-panel").classList.toggle("hidden");
});


これでクリックするとパネルが表示/非表示。

レイヤー情報をPixiコンテナと同期

レイヤー追加時に new PIXI.Container() を作ってステージに追加。

同時にパネルにも <div class="layer">新規レイヤー</div> を追加。

選択中のレイヤーは .active クラスを付けて見た目で分かるように。

削除・非表示機能の実装

削除 → container.destroy() してパネルからも消す。

非表示 → container.visible = false; を切り替え。

=====================================


マウス操作では Space+ドラッグ → スムーズに移動できる

ペン操作では「浮いた状態」でも動いてしまう（押し付けてなくてもホバーで動く感じ）

本来は「ペンがキャンバスに接触してから（pointerdown → move）」じゃないと移動が始まってほしくない

🔍 原因

これは PointerEvent の扱いが原因です。

マウスとペンの違い

マウス → mousedown / mousemove / mouseup

ペン → pointerdown / pointermove / pointerup

ペンは hover 時にも pointermove が飛ぶ（特に Windows Ink やタブレット環境）。

今のコードは pointermove をそのまま「カメラ移動」に結びつけているので、押していなくても動く。

判定不足

「Space が押されている」だけで移動を開始している。

「pointerdown（押している状態）」を確認していない。

結果、hover状態でも move が発火して動いてしまう。

✅ 改善策

pointerdown があるときだけ移動モードにするフラグを追加すれば直ります。

例：修正イメージ
let isPanning = false;
let isSpacePressed = false;

// Spaceキー押下状態を記録
document.addEventListener("keydown", (e) => {
  if (e.code === "Space") isSpacePressed = true;
});
document.addEventListener("keyup", (e) => {
  if (e.code === "Space") isSpacePressed = false;
  isPanning = false; // Space離したら強制終了
});

// pointerdownでフラグをオン
canvas.addEventListener("pointerdown", (e) => {
  if (isSpacePressed) {
    isPanning = true;
    lastX = e.clientX;
    lastY = e.clientY;
  }
});

// pointermoveで移動
canvas.addEventListener("pointermove", (e) => {
  if (isPanning && isSpacePressed) {
    const dx = e.clientX - lastX;
    const dy = e.clientY - lastY;
    camera.x += dx;
    camera.y += dy;
    lastX = e.clientX;
    lastY = e.clientY;
  }
});

// pointerupで終了
canvas.addEventListener("pointerup", () => {
  isPanning = false;
});

📝 改修ポイント

isSpacePressed だけでなく 「押しているかどうか」(pointerdown) を確認するフラグが必要。

これで **マウスもペンも「押し込み + Space」でのみ移動」**になる。

hover状態での pointermove は無視される。
