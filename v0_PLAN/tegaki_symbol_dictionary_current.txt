#### 🔧 座標ズレ問題の詳細
```javascript
// 問題メソッド: getCanvasCoordinates(e)
// 症状: カーソル位置から約200px右下で描画される
// 原因候補:
// 1. Canvas DOM配置とPixiJS内部座標の不一致
// 2. devicePixelRatio・スケール計算の誤用
// 3. getBoundingClientRect()の結果とCanvas実座標の乖離
// 4. レイヤー座標オフセット

// 現在の実装（app-core.js 362-397行）:
const canvas = this.pixiApp.view;
const rect = canvas.getBoundingClientRect();
const x = e.clientX - rect.left;  // 相対座標計算
const y = e.clientY - rect.top;
// 複雑なスケール変換処理（問題の可能性）
```
