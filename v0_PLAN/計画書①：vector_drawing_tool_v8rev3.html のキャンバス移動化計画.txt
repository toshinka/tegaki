=== 計画名 ===
vector_drawing_tool_v8rev3.html のキャンバス移動化計画（Pixi Container方式・ショートカット運用版）

=== 改修目的 ===
- PixiJS Container構造を前提にしたカメラ移動機能を実装
- UIボタンは設置せず、ショートカットキー操作のみで移動可能にする
- Homeキーでカメラ位置を初期化（リセット）
- canvasWidth / canvasHeight undefined エラーを回避する

=== 実装方針 ===
1. Container構造
   - stage
     └─ cameraContainer（マスク付き、表示窓）
         └─ worldContainer（描画対象レイヤーをまとめる）
     └─ uiContainer（UI表示用。マスク無し）

2. Camera管理
   - worldContainer.position.x / y でカメラ移動を実現
   - scale / rotation は将来用に予約（現時点では未使用）
   - reset() メソッドで (0,0) に戻す

3. 入力処理（ショートカット運用）
   - Space + ドラッグ：cameraContainer上のドラッグで worldContainer を移動
   - Space + Arrowキー：一定量移動
   - Homeキー：worldContainer を初期位置にリセット

4. マスク／カメラ枠
   - デフォルト 400x400
   - サイズ変更時は cameraContainer.mask を再生成
   - app.renderer.resize を呼ぶ

5. エラー対策
   - canvasWidth / canvasHeight は使わず、canvas.width / canvas.height を参照
   - undefined 参照を避ける

6. 拡張性
   - 将来的に拡縮、回転、反転操作を追加可能
   - レイヤー追加・UIとの分離も容易

💻 参考パッチ案（Pixi Container方式）

const app = new PIXI.Application({ width: 400, height: 400 });
document.body.appendChild(app.view);

// -------------------------
// Container構造
// -------------------------
const cameraContainer = new PIXI.Container();
const worldContainer = new PIXI.Container();
cameraContainer.addChild(worldContainer);
app.stage.addChild(cameraContainer);

const uiContainer = new PIXI.Container();
app.stage.addChild(uiContainer);

// -------------------------
// Camera管理（ショートカット運用）
// -------------------------
let isPanning = false;
let lastX = 0, lastY = 0;

cameraContainer.interactive = true;
cameraContainer.on('pointerdown', (e) => {
  if (e.data.originalEvent.getModifierState("Space")) {
    isPanning = true;
    const pos = e.data.global;
    lastX = pos.x;
    lastY = pos.y;
  }
});

cameraContainer.on('pointermove', (e) => {
  if (isPanning) {
    const pos = e.data.global;
    const dx = pos.x - lastX;
    const dy = pos.y - lastY;
    worldContainer.x += dx;
    worldContainer.y += dy;
    lastX = pos.x;
    lastY = pos.y;
    app.render();
  }
});

cameraContainer.on('pointerup', () => { isPanning = false; });
cameraContainer.on('pointerupoutside', () => { isPanning = false; });

// -------------------------
// キーボード操作
// -------------------------
document.addEventListener("keydown", (e) => {
  const step = 20;
  if (e.getModifierState("Space")) {
    switch (e.code) {
      case "ArrowLeft": worldContainer.x += step; break;
      case "ArrowRight": worldContainer.x -= step; break;
      case "ArrowUp": worldContainer.y += step; break;
      case "ArrowDown": worldContainer.y -= step; break;
    }
    app.render();
  }

  if (e.code === "Home") {
    worldContainer.x = 0;
    worldContainer.y = 0;
    app.render();
  }
});
