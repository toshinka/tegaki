# 🎯 キャンバス境界描画問題改修計画書 Phase1.2

## 📋 問題概要・影響範囲

### 🚨 現在の問題状況
- **問題**: キャンバス境界外からマウス/ペンを引いてキャンバス内に入っても描画開始されない
- **原因**: PixiJS hitAreaによる厳密な境界制限・PointerEvent境界越え処理未対応
- **影響**: エッジ付近での描画作業の利便性大幅低下・Clip Studio Paint水準に未達
- **緊急度**: 中（利便性向上・ユーザビリティ問題）

### 🎯 技術課題分析
```javascript
影響コンポーネント:
📝 pen-tool.js: startDrawing()・continueDrawing()境界判定
📝 eraser-tool.js: 同様の境界判定問題  
📝 canvas-manager.js: キャンバス領域・イベント処理・ヒットエリア設定
📝 tool-manager.js: ツールイベント統合・境界管理
📝 app-core.js: PixiJSイベント・PointerEvent統合

技術制約:
1. PixiJS hitAreaによる厳密な境界制限
2. mousedown境界外→mousemove境界内の未対応
3. PointerEvent・TouchEventの境界越え処理不完全
4. 描画継続性ロジック・境界判定統一不足
```

### STEP1: 境界イベント処理システム拡張
**対象**: `canvas-manager.js`
**期間**: 1日
**優先度**: 最高

```javascript
// 境界越えイベント処理システム
class BoundaryEventHandler {
    constructor(canvasManager) {
        this.canvas = canvasManager;
        this.config = ConfigManager.get('canvas.boundary');
        this.isTrackingOutside = false;
        this.outsidePointer = null;
    }
    
    initializeBoundaryTracking() {
        // document全体でPointer追跡
        document.addEventListener('pointerdown', this.handleGlobalPointerDown.bind(this));
        document.addEventListener('pointermove', this.handleGlobalPointerMove.bind(this));
        document.addEventListener('pointerup', this.handleGlobalPointerUp.bind(this));
        
        // 境界マージン設定
        this.boundaryMargin = this.config.margin || 10;
        this.expandedHitArea = this.createExpandedHitArea();
    }
    
    handleGlobalPointerDown(event) {
        const canvasRect = this.canvas.canvasElement.getBoundingClientRect();
        const isInsideCanvas = this.isPointInCanvas(event.clientX, event.clientY, canvasRect);
        
        if (!isInsideCanvas) {
            // キャンバス外でのPointerDown検出
            this.isTrackingOutside = true;
            this.outsidePointer = {
                id: event.pointerId,
                startX: event.clientX,
                startY: event.clientY,
                tool: this.canvas.toolManager.currentTool
            };
            
            EventBus.safeEmit('boundary.outside.start', {
                pointer: this.outsidePointer,
                canvasRect
            });
        }
    }
    
    handleGlobalPointerMove(event) {
        if (!this.isTrackingOutside || event.pointerId !== this.outsidePointer?.id) {
            return;
        }
        
        const canvasRect = this.canvas.canvasElement.getBoundingClientRect();
        const isInsideCanvas = this.isPointInCanvas(event.clientX, event.clientY, canvasRect);
        
        if (isInsideCanvas) {
            // キャンバス外→内への移行検出
            this.handleBoundaryCrossIn(event, canvasRect);
        }
    }
    
    handleBoundaryCrossIn(event, canvasRect) {
        // キャンバス座標変換
        const canvasX = event.clientX - canvasRect.left;
        const canvasY = event.clientY - canvasRect.top;
        
        // 描画開始イベント発火
        EventBus.safeEmit('boundary.cross.in', {
            tool: this.outsidePointer.tool,
            position: { x: canvasX, y: canvasY },
            pressure: event.pressure || 0.5,
            pointerId: event.pointerId
        });
        
        // 描画ツールに直接通知
        if (this.outsidePointer.tool) {
            this.outsidePointer.tool.handleBoundaryCrossIn(canvasX, canvasY, event);
        }
        
        this.isTrackingOutside = false;
        this.outsidePointer = null;
    }
}
```

```javascript
handleBoundaryCrossIn(x, y, pointerEvent) {
    if (!this.isActive) return;

        // 境界越え描画開始

}
```

```javascript
// PixiJS統合境界システム
initializeExpandedEventSystem() {
    // 拡張ヒットエリア設定
    const margin = ConfigManager.get('canvas.boundary.margin') || 20;
    
    this.app.stage.hitArea = new PIXI.Rectangle(
        -margin,
        -margin, 
        this.width + margin * 2,
        this.height + margin * 2
    );
    
    // インタラクティブ有効化
    this.app.stage.interactive = true;
    
    // 境界越えイベント統合
    this.setupBoundaryEventIntegration();
}

setupBoundaryEventIntegration() {
    // 境界越えイベント監視
    EventBus.on('boundary.cross.in', (data) => {
        this.handleCanvasBoundaryCross(data);
    });
    
    // PixiJSイベントと統一システム連携
    this.app.stage.on('pointerdown', (event) => {
        this.handlePixiPointerDown(event);
    });
    
    this.app.stage.on('pointermove', (event) => {
        this.handlePixiPointerMove(event);
    });
}

handleCanvasBoundaryCross(data) {
    // PixiJS座標系変換
    const globalPoint = new PIXI.Point(data.position.x, data.position.y);
    const localPoint = this.app.stage.toLocal(globalPoint);
    
    // 描画ツールに座標転送
    if (data.tool && typeof data.tool.handleBoundaryCrossIn === 'function') {
        data.tool.handleBoundaryCrossIn(localPoint.x, localPoint.y, {
            pressure: data.pressure,
            pointerId: data.pointerId
        });
    }
    
    console.log(`🎯 PixiJS境界越え処理: (${localPoint.x}, ${localPoint.y})`);
}
```
