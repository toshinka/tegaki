## 🎨 レイヤー構造（Phase1レイヤー分離版）

### レイヤー分離構造
```javascript
// layer0: 背景レイヤー（固定・消去対象外）
const backgroundLayer = {
    id: 'layer0',
    zIndex: 0,
    type: 'background',
    protected: true,           // 消去・クリア禁止
    description: 'キャンバス背景専用'
};

// layer1: 描画レイヤー（アクティブ・消去対象）
const drawingLayer = {
    id: 'layer1', 
    zIndex: 1,
    type: 'drawing',
    protected: false,          // 描画・消去可能
    description: 'ペン描画・消しゴム対象'
};
```

### レイヤー操作ルール
```javascript
// ✅ 許可される操作
canvasManager.setActiveLayer('layer1');        // 描画レイヤーをアクティブ
canvasManager.addGraphicsToLayer(graphics);    // アクティブレイヤーに追加
canvasManager.clearLayer('layer1');            // 描画レイヤーのクリア
canvasManager.eraseFromActiveLayer(graphics);  // アクティブレイヤーから消去

// 🚫 禁止される操作
canvasManager.setActiveLayer('layer0');        // 背景レイヤーはアクティブ禁止
canvasManager.clearLayer('layer0');            // 背景レイヤーのクリア禁止
```

## 🧹 消去機能（ERASE blend mode）

### 真の消去システム
```javascript
// EraserTool の消去処理
const eraseGraphics = new PIXI.Graphics();
eraseGraphics.blendMode = PIXI.BLEND_MODES.ERASE;  // 真の消去モード
eraseGraphics.lineStyle(eraserSize, 0x000000);     // 色は関係なし
eraseGraphics.drawCircle(x, y, eraserSize / 2);    // 消去範囲

// アクティブレイヤーに追加（消去実行）
activeLayer.addChild(eraseGraphics);
```

### 消去保護システム
```javascript
// 背景レイヤー保護
if (activeLayerId === 'layer0') {
    console.warn('⚠️ 背景レイヤーの消去は禁止されています');
    return; // 消去処理中止
}
```
