# JavaScript + PixiJS v7→v8 Rulebook v4.0 - 統一システム対応版

## 1. 基本理念（統一システム統合対応版）

### 1.1 核心方針
- **統一システム設計**: ConfigManager・ErrorManager・StateManager・EventBusによる統一アーキテクチャ
- **循環依存排除**: EventBus駆動による疎結合設計・依存関係最適化
- **段階的品質向上**: v7完成→v8移行の安定的発展戦略
- **node_modules統合**: CDN依存からローカル依存への安定移行・CORS制限回避
- **非破壊的変形**: ベクターベース変形によるアニメ制作品質確保
- **AI協働開発**: 統一システムによる効率的開発・保守

### 1.2 実装原則（統一システム対応版）
- **SOLID・DRY原則徹底**: 統一システムによる重複コード完全排除
- **統一システム活用**: 全設定値・エラー処理・状態管理の一元化
- **500行ファイル制限**: 統一システム分離により機能別最適化
- **責任分界明確化**: Manager・SystemレベルでのInterface統一
- **非破壊品質保証**: アニメ制作における品質劣化完全防止

### 1.3 Phase移行戦略
- **Phase1**: HTML単体→統一システム分割移行（v7基盤）
- **Phase2**: 描画機能完成・レイヤーシステム（v7完成版）  
- **Phase3**: アニメーション・GIF機能（v7最終版）
- **Phase4**: PixiJS v8移行・120FPS・SPINE対応

## 5. 統一システム活用パターン

### 5.1 統一初期化パターン
```javascript
// FutabaDrawingTool統一初期化（main.js）
class FutabaDrawingTool {
    async init() {
        try {
            // Step 1: 統一システム確認
            await this.validateUnifiedSystems();
            
            // Step 2: 設定値読み込み
            this.config = ConfigManager.getApplicationConfig();
            
            // Step 3: AppCore初期化
            await this.initializeAppCore();
            
            // Step 4: 統一イベント発行
            EventBus.emit(EventBus.Events.INITIALIZATION_COMPLETED, {
                version: this.version,
                initTime: performance.now() - this.startTime
            });
            
        } catch (error) {
            ErrorManager.showError('critical', error, {
                additionalInfo: 'AppCore初期化失敗'
            });
        }
    }
    
    validateUnifiedSystems() {
        const required = ['ConfigManager', 'ErrorManager', 'StateManager', 'EventBus'];
        const missing = required.filter(sys => !window[sys]);
        
        if (missing.length > 0) {
            throw new Error(`統一システム未読み込み: ${missing.join(', ')}`);
        }
    }
}
```

### 5.3 統一エラー処理パターン
```javascript
// 統一エラー処理活用例
class DrawingToolSystem {
    setBrushSize(size) {
        try {
            // 妥当性確認（統一）
            if (!ConfigManager.validate('drawing.pen.defaultSize', size)) {
                ErrorManager.showError('warning', 
                    `無効なブラシサイズ: ${size}`, 
                    { duration: 5000 });
                return;
            }
            
            this.brushSize = size;
            
            // イベント発行（循環依存排除）
            EventBus.emit(EventBus.Events.BRUSH_SIZE_CHANGED, { 
                size, 
                tool: this.currentTool 
            });
            
        } catch (error) {
            ErrorManager.showError('error', error, {
                additionalInfo: `ブラシサイズ設定失敗: ${size}`
            });
        }
    }
}

class UIController {
    init() {
        // イベントリスナー登録（循環依存なし）
        EventBus.on(EventBus.Events.BRUSH_SIZE_CHANGED, (data) => {
            this.updateSizePresets();
            this.updateSliderValue('pen-size-slider', data.size);
        });
        
        EventBus.on(EventBus.Events.ERROR_OCCURRED, (error) => {
            this.handleUIError(error);
        });
    }
}
```

### 5.4 統一状態管理パターン
```javascript
// デバッグ・監視での統一状態利用
class PerformanceMonitor {
    checkPerformanceWarnings() {
        const appState = StateManager.getApplicationState();
        const warnings = [];
        
        // FPS警告
        if (appState.performance.currentFPS < 30) {
            warnings.push(`低FPS: ${appState.performance.currentFPS}`);
        }
        
        // メモリ警告
        if (appState.performance.memoryUsage?.used > 100) {
            warnings.push(`高メモリ使用量: ${appState.performance.memoryUsage.used}MB`);
        }
        
        // エラー数警告
        if (appState.errors.totalAppErrors > 5) {
            warnings.push(`エラー多発: ${appState.errors.totalAppErrors}件`);
        }
        
        if (warnings.length > 0) {
            EventBus.emit(EventBus.Events.PERFORMANCE_WARNING, warnings);
        }
    }
}

// 健全性チェック活用
function debugApplicationHealth() {
    const health = StateManager.healthCheck();
    
    if (!health.healthy) {
        ErrorManager.showError('warning', 
            `アプリケーション健全性問題: ${health.issues.join(', ')}`,
            { showDebug: true });
    }
    
    console.log('🏥 健全性チェック結果:', health);
    return health;
}
```

## 8. 禁止事項（統一システム強化版）

### 8.1 統一システム固有禁止事項
- **統一システム迂回**: ConfigManager等を使わない直接設定値参照禁止
- **重複エラー処理**: ErrorManager を使わない独自エラー表示禁止
- **状態管理重複**: StateManager を使わない独自状態取得禁止
- **直接通信**: EventBus を使わないクラス間直接通信禁止（循環依存発生）

### 8.2 統一システム実装禁止パターン
```javascript
// ❌ 禁止: 統一システム迂回
const brushSize = 16.0;  // ConfigManager.get('drawing.pen.defaultSize') を使用

// ❌ 禁止: 独自エラー表示
alert('エラー発生');     // ErrorManager.showError() を使用

// ❌ 禁止: 直接クラス通信
uiController.updateSlider();  // EventBus経由で通信

// ❌ 禁止: 独自状態取得
const state = { tool: this.currentTool };  // StateManager.getDrawingState() を使用
```

### 8.3 基本禁止事項（継続）
- **ESM/TypeScript混在**: Pure JavaScript維持・type="module"禁止
- **CDN依存**: node_modules使用可能ライブラリのCDN使用禁止
- **循環依存**: ファイル間循環依存の完全禁止（EventBus活用）
- **ラスター変換**: ベクターデータの破壊的変換禁止

```javascript
// 統一システム経由でのv8対応
class V8Compatibility {
    static async createApplication(options) {
        const v8Config = ConfigManager.getV8Config();
        
        try {
            if (v8Config.enabled && PIXI.VERSION.startsWith('8')) {
                // v8移行モード
                EventBus.emit(EventBus.Events.V8_MIGRATION_STARTED);
                return await PIXI.Application.init(options);
            } else {
                // v7互換モード
                return new PIXI.Application(options);
            }
        } catch (error) {
            ErrorManager.showError('v8-migration', error, {
                additionalInfo: 'v8移行処理失敗'
            });
            throw error;
        }
    }
}
```
