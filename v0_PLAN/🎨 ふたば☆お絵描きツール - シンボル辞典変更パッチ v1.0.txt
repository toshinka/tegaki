# 🎨 ふたば☆お絵描きツール - シンボル辞典変更パッチ v1.0

> **適用対象**: 2_tegaki-symbol-dictionary.txt  
> **変更理由**: 統一計画書に基づく重複関数統合・設定値一元化・命名規則統一  
> **実装日**: 2025年8月18日  

## 📋 変更概要

統一計画書に従い、以下の統一システムを追加し、既存の重複関数を統合しました：

1. **ConfigManager**: 設定値一元化
2. **ErrorManager**: エラー処理統一  
3. **StateManager**: 状態管理統一
4. **EventBus**: 循環依存排除・イベント駆動アーキテクチャ

### 🔄 循環依存の排除（EventBus活用）

```javascript
// ❌ 旧: 循環依存あり
AppCore → DrawingToolSystem → AppCore (参照)
UIController → DrawingToolSystem → UIController (イベント)

// ✅ 新: EventBus経由で循環依存排除
DrawingToolSystem → EventBus.emit('brushSizeChanged', data)
UIController → EventBus.on('brushSizeChanged', callback)

// 実装例
class DrawingToolSystem {
    setBrushSize(size) {
        this.brushSize = size;
        // 直接UIControllerを呼ばずにイベント発行
        EventBus.emit(EventBus.Events.BRUSH_SIZE_CHANGED, { size });
    }
}

class UIController {
    init() {
        // イベント監視でUI更新
        EventBus.on(EventBus.Events.BRUSH_SIZE_CHANGED, ({ size }) => {
            this.updateSizePresets();
            this.updateSliderValue('pen-size-slider', size);
        });
    }
}
```
