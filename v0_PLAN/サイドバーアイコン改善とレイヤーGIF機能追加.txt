# Phase1後期改修計画書 - サイドバーアイコン改善とレイヤー/GIF機能追加

## 🎯 改修概要

### 現状問題分析
1. **@tabler/icons問題**: 現在`@tabler/icons-react`を使用しているがPure JavaScript環境では直接利用不可
2. **アイコン不統一**: サイドバーのアイコンがSVG直書きで統一性に欠ける
3. **機能アイコン不足**: レイヤー・GIF機能のアイコンが設定の上にない

### 目標達成状況
- DRY・SOLID原則の維持
- Pure JavaScript環境での@tabler/icons活用
- 新機能追加（レイヤー・GIF）
- 視覚的統一性の向上

## 🔧 技術実装方針

### アイコンライブラリ統合方式
```javascript
// @tabler/icons Pure JavaScript活用パターン
class TablerIconManager {
    constructor() {
        this.iconCache = new Map();
        this.baseUrl = './node_modules/@tabler/icons/icons/';
    }
    
    async getIcon(iconName) {
        if (this.iconCache.has(iconName)) {
            return this.iconCache.get(iconName);
        }
        
        try {
            const response = await fetch(`${this.baseUrl}${iconName}.svg`);
            const svgText = await response.text();
            this.iconCache.set(iconName, svgText);
            return svgText;
        } catch (error) {
            ErrorManager.showError('icon-load', 
                `アイコン読み込み失敗: ${iconName}`, { iconName });
            return null;
        }
    }
}
```

#### レイヤー（layers）
