## 📋 発見された問題の分類

### 🚨 重大問題（AI協働開発を阻害）

#### 1. **命名の曖昧性・重複**
- 同じ機能に複数のアクセス方法が存在
- 似た名前で異なる機能
- 状態管理の責任分界が不明確

#### 2. **設定値の分散・重複**
- 同じ値が複数箇所で定義
- デフォルト値の不整合
- 設定変更の影響範囲が不明

#### 3. **初期化の複雑性**
- 初期化シーケンスが複雑すぎる
- エラーハンドリングが分散
- 状態確認方法が統一されていない

## 🎯 改修方針

### 📐 **統一原則**

1. **Single Source of Truth**: 設定値・状態情報は1箇所に集約
2. **Unified API**: 同じ機能は1つのAPIでのみ提供
3. **Clear Responsibility**: 各クラスの責任を明確に分離
4. **Predictable Naming**: 命名規則を統一・予測可能にする

### 🏗️ **アーキテクチャ改善**

1. **ConfigManager**: 全設定値を一元管理
2. **StateManager**: アプリケーション状態を一元管理
3. **APIGateway**: 外部アクセス用の統一インターフェース
4. **ErrorManager**: エラーハンドリングを一元管理

### Phase1.5-Step2: API統一

#### 🎯 **APIGateway クラス新設**

```javascript
// js/api-gateway.js - 外部アクセス用統一インターフェース
class APIGateway {
    constructor(futabaDrawingTool) {
        this.app = futabaDrawingTool
    }
    
    // 🎨 ツール制御（統一）
    selectTool(toolName) {
        // UI更新・イベント発火・状態管理を含む統一処理
        return this.app.appCore.uiController.setTool(toolName)
    }
    
    // 🎚️ ブラシ設定（統一）
    setBrushSize(size) {
        // ToolSystem更新・UI更新・設定保存を含む統一処理
        this.app.appCore.toolSystem.setBrushSize(size)
        this.app.appCore.uiController.updateSliderValue('pen-size-slider', size)
        this.app.appCore.uiController.updateSizePresets()
    }
    
    // 📐 キャンバス制御（統一）
    resizeCanvas(width, height, centerContent = false) {
        return this.app.appCore.resize(width, height, centerContent)
    }
    
    // 📊 状態取得（統一）
    getState() {
        return this.app.getAppState()
    }
    
    // 🔍 診断・デバッグ（統一）
    diagnose() {
        return {
            dependencies: this.checkDependencies(),
            performance: this.getPerformanceStats(),
            errors: this.getErrorLog()
        }
    }
}
```

### Phase1.5-Step3: エラーハンドリング統一

#### 🎯 **ErrorManager クラス新設**

```javascript
// js/managers/error-manager.js
class ErrorManager {
    constructor() {
        this.errorLog = []
        this.errorTypes = {
            DEPENDENCY: 'dependency',
            INITIALIZATION: 'initialization',
            RUNTIME: 'runtime',
            UI: 'ui'
        }
    }
    
    // 統一エラー処理
    handleError(error, type = 'runtime', context = '') {
        const errorInfo = this.createErrorInfo(error, type, context)
        this.errorLog.push(errorInfo)
        
        switch(errorInfo.severity) {
            case 'critical':
                this.showCriticalError(errorInfo)
                break
            case 'warning':
                this.showWarning(errorInfo)
                break
            default:
                this.showError(errorInfo)
        }
    }
    
    // エラー表示統一（1つの関数で全パターン対応）
    showError(errorInfo) {
        const errorDiv = this.createErrorElement(errorInfo)
        this.attachErrorElement(errorDiv, errorInfo)
        this.scheduleErrorRemoval(errorDiv, errorInfo)
    }
    
    // 診断機能統一
    diagnoseError(error) {
        return {
            error: error.message,
            type: this.detectErrorType(error),
            recommendations: this.getRecommendations(error),
            relatedIssues: this.findRelatedIssues(error)
        }
    }
}
```

### Phase1.5-Step4: 初期化簡素化

#### 🎯 **InitializationManager クラス新設**

```javascript
// js/managers/initialization-manager.js
class InitializationManager {
    constructor(configManager, errorManager) {
        this.config = configManager
        this.errors = errorManager
        this.steps = new Map()
        this.currentStep = null
    }
    
    // 初期化ステップ登録（宣言的）
    registerStep(name, dependencies, handler) {
        this.steps.set(name, { dependencies, handler, completed: false })
    }
    
    // 依存関係解決付き実行
    async execute() {
        const sortedSteps = this.sortStepsByDependencies()
        
        for (const step of sortedSteps) {
            try {
                this.currentStep = step.name
                await step.handler()
                step.completed = true
            } catch (error) {
                this.errors.handleError(error, 'initialization', step.name)
                throw new Error(`初期化ステップ '${step.name}' で失敗: ${error.message}`)
            }
        }
    }
    
    // 初期化状態確認統一
    getInitializationStatus() {
        return {
            currentStep: this.currentStep,
            completedSteps: Array.from(this.steps.entries())
                .filter(([_, step]) => step.completed)
                .map(([name]) => name),
            failedSteps: this.errors.getErrorsByType('initialization'),
            progress: this.calculateProgress()
        }
    }
}
```
