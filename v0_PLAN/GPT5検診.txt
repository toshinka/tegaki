包括的改修計画（v8 phase1rev20）
*************************************

■ 問題の概要
- エラー: this.appCore.createCanvasV8 is not a function
- 原因: app-core.js に createCanvasV8() が未実装、または名前空間公開されていない
- 結果: 
  ・キャンバス生成が失敗 → ペン描画できない
  ・サイドバーアイコンも初期化されない（ToolManager 未登録）

■ 根本原因
1. tegaki-application.js は this.appCore.createCanvasV8() を呼ぶが、app-core.js に定義なし
2. CanvasManager の create メソッドを呼び出すべきなのに、中間レイヤー(appCore)が橋渡し未実装
3. Manager / Tool の初期化順が曖昧
   - Bootstrap → TegakiApplication → AppCore → Manager の流れが崩れている
4. 名前空間公開不足
   - window.Tegaki 内に公開が揃っていない

■ 正しい初期化フロー
1. index.html  
   → Bootstrap.start()
2. bootstrap.js  
   → new TegakiApplication().initialize()
3. tegaki-application.js  
   → this.appCore.initialize()  
   → this.appCore.createCanvasV8() （キャンバス生成）  
   → Manager登録 (CanvasManager, ToolManager)  
   → Sidebarアイコン生成
4. app-core.js  
   → createCanvasV8() 内で CanvasManager.createCanvas() を呼ぶ  
   → window.Tegaki.CanvasManagerInstance に保持
5. canvas-manager.js  
   → 実際の <canvas> DOM 要素を生成し Pixi.Application に接続
6. tool-manager.js  
   → PenTool / EraserTool を登録し、サイドバーアイコンを生成
7. pen-tool.js / eraser-tool.js  
   → Tool の描画処理を提供

■ 包括的改修計画
1. app-core.js に createCanvasV8() を追加  
   - 内部で CanvasManagerInstance.createCanvas(width, height) を呼ぶ  
   - デフォルトサイズ 400x400  
   - window.Tegaki 名前空間に公開
2. tegaki-application.js の createCanvasV8() 呼び出しをそのまま活かす  
   - ただし AppCore に確実に橋渡し実装があることを前提とする
3. manager 系ファイルに @manager-key を付与し、window.Tegaki 内のキーと一致させる  
   - 例: window.Tegaki.CanvasManagerInstance
4. tool-manager.js 初期化時に PenTool/EraserTool を登録  
   - アイコンも icons.js から取得して表示
5. 全ファイルにヘッダーコメントを追加（以下必須タグ入り）  
   - @provides / @uses / @initflow / @forbids / @manager-key / @dependencies-strict / @integration-flow / @method-naming-rules
6. Manager系には @error-handling / @testing-hooks を追加  
   Tool系には @state-management / @performance-notes を追加
7. エラーハンドリング強化  
   - createCanvasV8() が失敗したら ErrorManager で捕捉し、Bootstrap 側に伝搬

■ 改修後の呼び出し例
- index.html  
  → Bootstrap.start()  
- bootstrap.js  
  → app.initialize()  
- tegaki-application.js

- canvas-manager.js

- user操作  
  → サイドバーからペン選択 → PenTool が Canvas に描画 ✅
