## 📋 統一システムクラス

### ConfigManager
**ファイル**: `js/utils/config-manager.js`  
**責任**: 設定値一元管理・デフォルト値定義  
**依存**: なし  

```javascript
// メソッド
ConfigManager.get(path)                    // 設定値取得
ConfigManager.has(path)                    // 設定値存在確認
ConfigManager.getNestedValue(obj, path)    // ネスト値取得
ConfigManager.getCanvasConfig()            // キャンバス設定取得
ConfigManager.getDrawingConfig(tool)       // 描画ツール設定取得
ConfigManager.getColors()                  // ふたばカラーパレット取得

// 設定構造
defaultConfig = {
    canvas: { width, height, backgroundColor },
    drawing: { pen, eraser },
    ui: { slider, popup, keyboard },
    performance: { targetFPS, updateInterval },
    colors: { futabaMaroon, futabaCream, ... }
}
```

### ErrorManager
**ファイル**: `js/utils/error-manager.js`  
**責任**: エラー表示統一・エラーレベル管理  
**依存**: ConfigManager  

```javascript
// メソッド
ErrorManager.showError(type, error, options)     // エラー表示統一
ErrorManager.showWarningMessage(error, options)  // 警告表示
ErrorManager.showCriticalError(error, options)   // 致命的エラー
ErrorManager.showRecoveryMessage(error, options) // 回復メッセージ
ErrorManager.hasErrors()                         // エラー存在確認
ErrorManager.getErrorLog()                       // エラーログ取得

// エラータイプ
'warning'   // 警告レベル
'error'     // エラーレベル  
'critical'  // 致命的レベル
'recovery'  // 回復レベル
```

### StateManager
**ファイル**: `js/utils/state-manager.js`  
**責任**: 状態情報統一取得・デバッグ情報構造化  
**依存**: なし  

```javascript
// メソッド
StateManager.getApplicationState()    // アプリケーション状態
StateManager.getCoreState()           // コア状態
StateManager.getUIState()             // UI状態
StateManager.getDrawingState()        // 描画状態
StateManager.getPerformanceState()    // パフォーマンス状態
StateManager.getErrorState()          // エラー状態
StateManager.healthCheck()            // 健全性チェック

// グローバルデバッグ関数
window.getState()     // StateManager.getApplicationState()
window.getCoreState()    // StateManager.getCoreState()
window.getDrawingState() // StateManager.getDrawingState()
```

### EventBus
**ファイル**: `js/utils/event-bus.js`  
**責任**: イベント駆動・循環依存排除  
**依存**: なし  

```javascript
// メソッド
EventBus.emit(eventType, data)      // イベント発行
EventBus.on(eventType, callback)    // イベント監視
EventBus.off(eventType, callback)   // イベント解除

// イベントタイプ例
'brushSizeChanged'    // ブラシサイズ変更
'toolChanged'         // ツール変更
'canvasResized'       // キャンバスリサイズ
'uiStateChanged'      // UI状態変更
```

## ⚙️ 統一設定値・定数一覧

### 統一設定値（ConfigManager.defaultConfig）
```javascript
// キャンバス設定（canvas.*）
canvas.width: 400                      // デフォルト幅
canvas.height: 400                     // デフォルト高さ
canvas.backgroundColor: 0xf0e0d6       // ふたば背景色
canvas.maxWidth: 4096                  // 最大幅
canvas.maxHeight: 4096                 // 最大高さ

// PixiJS設定（pixi.*）
pixi.antialias: true                   // アンチエイリアス
pixi.resolution: 1                     // 解像度
pixi.autoDensity: false                // 自動密度

// 描画ツール設定（drawing.*）
drawing.pen.defaultSize: 16.0          // デフォルトブラシサイズ
drawing.pen.defaultColor: 0x800000     // ふたばマルーン
drawing.pen.defaultOpacity: 0.85       // デフォルト不透明度
drawing.pen.defaultPressure: 0.5       // デフォルト筆圧
drawing.pen.defaultSmoothing: 0.3      // デフォルト線補正
drawing.pen.minSize: 0.1               // 最小ブラシサイズ
drawing.pen.maxSize: 100.0             // 最大ブラシサイズ

// UI設定（ui.*）
ui.slider.updateThrottle: 16           // スライダー更新間隔（約60FPS）
ui.slider.precision.size: 1            // サイズ精度
ui.popup.animationDuration: 300        // ポップアップアニメーション

// パフォーマンス設定（performance.*）
performance.targetFPS: 60              // 目標FPS
performance.maxInitTime: 5000          // 最大初期化時間（ms）
performance.retryAttempts: 3           // リトライ回数
performance.retryDelay: 200            // リトライ間隔（ms）

// エラー設定（error.*）
error.displayDuration: 10000           // エラー表示時間（ms）
error.maxRetries: 3                    // 最大リトライ数
error.showDebugInfo: true              // デバッグ情報表示
```
