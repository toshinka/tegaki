🔍 原因候補（今回のソース構成から見て）

pen-tool.js 内の pointerdown → pointermove の初期処理で moveTo() をしていない。

ToolManager や CanvasManager が、ストローク開始位置を初期化する前に lineTo() を実行している。

lastX / lastY の初期値が 0 のまま、あるいは未定義扱いで lineTo されている。

✅ 修正案

pen-tool.js のストローク開始処理に 初期位置の moveTo を必ず入れるようにしてください。
例えばこんな感じになります：

// pen-tool.js 内
onPointerDown(event) {
    const pos = this.getCanvasPosition(event);

    this.drawing = true;
    this.lastX = pos.x;
    this.lastY = pos.y;

    this.graphics.moveTo(this.lastX, this.lastY); // 👈 ここを必ず呼ぶ
}

onPointerMove(event) {
    if (!this.drawing) return;

    const pos = this.getCanvasPosition(event);

    this.graphics.lineTo(pos.x, pos.y);
    this.graphics.moveTo(pos.x, pos.y); // 👈 スムーズに描くなら moveTo を更新
    this.lastX = pos.x;
    this.lastY = pos.y;
}

onPointerUp(event) {
    this.drawing = false;
}

⚡ 対策まとめ

pointerdown 時に moveTo(x,y) を入れる（これで (0,0) 起点の直線は出なくなる）。

lastX/lastY の初期値は null にして、null なら lineTo せず moveTo する。

if (this.lastX == null) {
    g.moveTo(x, y);
} else {
    g.lineTo(x, y);
}
this.lastX = x;
this.lastY = y;


pointerup で lastX/lastY をリセットしておく（次のストロークがまた (0,0) から始まらないように）。

👉 これで「左上から直線が伸びる」現象は確実に消えます。

pen-tool.js を確認した上で、左上から直線が伸びるバグを解消するパッチを作りました。
修正ポイントは「ストローク開始時に必ず moveTo を実行」「lastX/lastY をリセット」です。

🔧 修正版 pen-tool.js （パッチ）
// tools/pen-tool.js
class PenTool {
    constructor(app, graphics) {
        this.app = app;
        this.graphics = graphics;

        this.drawing = false;
        this.lastX = null;
        this.lastY = null;

        this.bindEvents();
    }

    bindEvents() {
        this.app.view.addEventListener("pointerdown", this.onPointerDown.bind(this));
        this.app.view.addEventListener("pointermove", this.onPointerMove.bind(this));
        this.app.view.addEventListener("pointerup", this.onPointerUp.bind(this));
        this.app.view.addEventListener("pointerupoutside", this.onPointerUp.bind(this));
    }

    getCanvasPosition(event) {
        const rect = this.app.view.getBoundingClientRect();
        return {
            x: event.clientX - rect.left,
            y: event.clientY - rect.top
        };
    }

    onPointerDown(event) {
        const pos = this.getCanvasPosition(event);

        this.drawing = true;
        this.lastX = pos.x;
        this.lastY = pos.y;

        // ✅ 初期位置を必ず moveTo
        this.graphics.moveTo(this.lastX, this.lastY);
    }

    onPointerMove(event) {
        if (!this.drawing) return;

        const pos = this.getCanvasPosition(event);

        if (this.lastX === null || this.lastY === null) {
            // ✅ 前回座標が無ければ lineTo せず moveTo
            this.graphics.moveTo(pos.x, pos.y);
        } else {
            this.graphics.lineTo(pos.x, pos.y);
        }

        this.lastX = pos.x;
        this.lastY = pos.y;
    }

    onPointerUp(event) {
        this.drawing = false;

        // ✅ 座標をリセット
        this.lastX = null;
        this.lastY = null;
    }
}

window.PenTool = PenTool;

📝 改修内容まとめ

onPointerDown

this.graphics.moveTo(x, y) を必ず呼び出し。

onPointerMove

lastX/lastY が無ければ moveTo、あれば lineTo。

onPointerUp

lastX/lastY を null にリセットして次回に備える。
