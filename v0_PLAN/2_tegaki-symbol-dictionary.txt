### ✏️ DrawingToolSystem（app-core.js内）

**責務**: 描画ツール管理・ブラシ制御・描画処理・元HTML版準拠の描画実装

```javascript
class DrawingToolSystem {
    constructor(appCore)
    
    // 🔧 描画エンジン管理
    initializeDrawingEngine()              // 描画エンジン初期化
    
    // 🔧 ツール管理
    setTool(tool)                          // ツール設定
    setBrushSize(size)                     // ブラシサイズ設定（0.1-100範囲）
    setOpacity(opacity)                    // 不透明度設定（0-1範囲）
    setPressure(pressure)                  // 筆圧設定（0-1範囲）
    setSmoothing(smoothing)                // 線補正設定（0-1範囲）
    
    // 🎨 描画制御（修復版）
    startDrawing(x, y)                     // 描画開始（修復版）
    continueDrawing(x, y)                  // 描画継続（修復版）
    stopDrawing()                          // 描画終了
    
    // 🖌️ パス作成（元HTML版準拠）
    createPenPath(x, y)                    // ペンパス作成（元HTML版準拠）
    createEraserPath(x, y)                 // 消しゴムパス作成
    drawLine(path, x1, y1, x2, y2)         // 線描画（元HTML版準拠の円形ブラシ方式）
    
    // ⚙️ 設定値（初期値）
    brushSize: 16.0                        // ブラシサイズ
    brushColor: 0x800000                   // ふたばマルーン
    opacity: 0.85                          // 不透明度
    pressure: 0.5                          // 筆圧
    smoothing: 0.3                         // 線補正
    
    // 🏷️ 状態プロパティ
    appCore                               // AppCore インスタンス参照
    currentTool                           // string - 現在のツール名
    isDrawing                             // boolean - 描画中フラグ
    currentPath                           // Object - 現在のパスオブジェクト
    lastPoint                             // Object - 最後の描画点
    extensionsAvailable                   // boolean - 拡張機能利用可能性
}
```
