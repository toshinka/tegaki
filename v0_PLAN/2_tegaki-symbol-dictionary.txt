# 🎨 ふたば☆お絵描きツール - シンボル辞典・API定義書 v1.0

> **作成目的**: AI協働開発での効率化・誤読防止・メンテナンス性向上  
> **対象Phase**: Phase1.4s1rev1（依存関係解決版）  
> **最終更新**: 2025年8月17日

## 📋 概要

このシンボル辞典は、ふたば☆お絵描きツールの全クラス・関数・設定値を網羅した技術リファレンスです。AIによるコード修正時の勘違いや依存関係エラーを防止し、効率的な開発を支援します。

## 🏗️ メインクラス定義

### 🎨 FutabaDrawingTool（main.js）

**責務**: アプリケーション全体の初期化・エラー処理・依存関係管理

```javascript
class FutabaDrawingTool {
    constructor()
    
    // 🎯 コア機能
    async init()                    // アプリケーション初期化
    async validateDependencies()    // 依存関係確認
    async initializeAppCore()       // AppCore統合
    
    // 🔧 初期化ステップ
    performPreInitializationChecks()    // DOM要素確認
    initializeExtensions()              // 拡張ライブラリ初期化
    setupUIIntegration()                // UI統合
    finalizeInitialization()            // 最終確認
    
    // 🛡️ エラー処理
    async attemptFallbackInitialization(error)  // フォールバック初期化
    showErrorMessage(error)                      // エラー表示
    showRecoveryMessage(originalError)           // 回復メッセージ
    showCriticalErrorMessage(originalError, fallbackError)  // 致命的エラー
    
    // 📊 状態管理
    getAppState()                   // 詳細状態取得
    startDebugMode()               // デバッグモード開始
    
    // ⚙️ 設定値（config プロパティ）
    config.maxInitTime             // 5000ms - 最大初期化時間
    config.retryAttempts           // 3 - リトライ回数
    config.retryDelay              // 200ms - リトライ間隔
    config.requiredDependencies    // ['PIXI', 'PixiExtensions', 'AppCore']
    
    // 🏷️ 状態プロパティ
    isInitialized                  // boolean - 初期化完了フラグ
    appCore                        // AppCore インスタンス
    initializationSteps            // Array - 完了した初期化ステップ
    errorLog                       // Array - エラーログ配列
}
```

### ✏️ DrawingToolSystem（app-core.js内）

**責務**: 描画ツール管理・ブラシ制御・描画処理

```javascript
class DrawingToolSystem {
    constructor(appCore)
    
    // 🎨 描画制御
    startDrawing(x, y)                   // 描画開始
    continueDrawing(x, y)                // 描画継続
    stopDrawing()                        // 描画終了
    
    // 🔧 ツール管理
    setTool(tool)                        // ツール設定
    setBrushSize(size)                   // ブラシサイズ設定
    setOpacity(opacity)                  // 不透明度設定
    setPressure(pressure)                // 筆圧設定
    setSmoothing(smoothing)              // 線補正設定
    
    // 🖌️ パス作成
    createPenPath(x, y)                  // ペンパス作成
    createEraserPath(x, y)               // 消しゴムパス作成
    drawLine(path, x1, y1, x2, y2)       // 線描画
    drawInitialPoint(path, x, y)         // 初期点描画
    
    // 🔢 ユーティリティ
    calculateDistance(x1, y1, x2, y2)    // 距離計算
    updatePressureMonitor()              // 筆圧モニター更新
    generatePathInfo(tool, x, y)         // パス基本情報生成
    
    // ⚙️ 設定値（config プロパティ）
    config.brush.size                    // 16.0 - ブラシサイズ
    config.brush.color                   // 0x800000 - ブラシ色
    config.brush.opacity                 // 0.85 - 不透明度
    config.brush.minSize                 // 0.1 - 最小サイズ
    config.brush.maxSize                 // 100.0 - 最大サイズ
    config.drawing.pressure              // 0.5 - 筆圧
    config.drawing.smoothing             // 0.3 - 線補正
    config.drawing.minDistance           // 1.5 - 最小描画距離
    
    // 🔧 設定値アクセサ
    get brushSize()                      // ブラシサイズ取得
    set brushSize(value)                 // ブラシサイズ設定
    get brushColor()                     // ブラシ色取得
    set brushColor(value)                // ブラシ色設定
    get opacity()                        // 不透明度取得
    set opacity(value)                   // 不透明度設定
    get pressure()                       // 筆圧取得
    set pressure(value)                  // 筆圧設定
    get smoothing()                      // 線補正取得
    set smoothing(value)                 // 線補正設定
    
    // 🏷️ 主要プロパティ
    appCore                             // AppCore インスタンス参照
    currentTool                         // string - 現在のツール名
    isDrawing                           // boolean - 描画中フラグ
    currentPath                         // Object - 現在のパスオブジェクト
    lastPoint                           // Object - 最後の描画点
    extensionsAvailable                 // boolean - 拡張機能利用可能性
}
```

class PerformanceMonitor {
    constructor(config)
    
    // 🚀 監視制御
    start()                            // 監視開始
    stop()                             // 監視停止
    startFPSMonitoring()               // FPS監視開始
    startMemoryMonitoring()            // メモリ監視開始
    
    // 📈 メトリクス更新
    updateFPSMetrics(fps)              // FPS指標更新
    displayFPS(fps)                    // FPS表示
    notifyCallbacks(type, value)       // コールバック通知
    
    // 📊 統計・警告
    getStats()                         // 統計情報取得
    checkPerformanceWarnings()         // パフォーマンス警告チェック
    onUpdate(callback)                 // 更新コールバック登録
    
    // ⚙️ 設定値（config プロパティ）
    config.targetFPS                   // 60 - 目標FPS
    config.updateInterval              // 1000ms - 更新間隔
    config.memoryCheckInterval         // 5000ms - メモリチェック間隔
    
    // 📊 メトリクス（metrics プロパティ）
    metrics.frameCount                 // フレーム数
    metrics.lastTime                   // 最終更新時間
    metrics.currentFPS                 // 現在のFPS
    metrics.averageFPS                 // 平均FPS
    metrics.minFPS                     // 最小FPS
    metrics.maxFPS                     // 最大FPS
    metrics.memoryUsage                // メモリ使用量（MB）
    
    // 🏷️ 状態プロパティ
    isRunning                          // boolean - 監視実行中フラグ
    updateCallbacks                    // Set - 更新コールバック群
}

## ⚠️ 重要な注意事項

### 🚫 避けるべき操作

```javascript
// ❌ 直接DOM操作（UIController を経由すること）
document.getElementById('pen-tool').click()

// ❌ 依存関係確認なしでの拡張機能使用
new PIXI.UI.Button()  // PixiExtensions.hasFeature('ui') で確認すること

// ❌ 初期化前のAppCore操作
appCore.resize(800, 600)  // isInitialized 確認が必要

// ❌ エラーハンドリングなしでの描画操作
appCore.toolSystem.startDrawing(x, y)  // try-catch で囲むこと
```

### ✅ 推奨パターン

```javascript
// ✅ 存在確認後の操作
if (window.futabaDrawingTool && window.futabaDrawingTool.isInitialized) {
    // 操作実行
}

// ✅ 拡張機能の安全な使用
if (window.PixiExtensions.hasFeature('ui')) {
    const Button = window.PixiExtensions.getComponent('ui', 'Button')
    // Button 使用
}

// ✅ エラーハンドリング付き操作
try {
    appCore.toolSystem.setBrushSize(newSize)
    appCore.uiController.updateSizePresets()
} catch (error) {
    console.error('ブラシサイズ設定エラー:', error)
}
```

### 🔧 トラブルシューティング

```javascript
// 初期化失敗時の確認手順
1. window.APP_INIT_MONITOR.checkAllDependencies()  // 依存関係確認
2. window.getAppState()                            // アプリ状態確認
3. console.log で各ステップの完了状況確認
4. ブラウザのDevToolsでネットワークエラー確認

// 描画不具合時の確認
1. appCore.app?.view が DOM に接続されているか
2. drawingContainer が存在するか
3. toolSystem が正常初期化されているか
4. エラーログ (futabaDrawingTool.errorLog) 確認

// UI操作不具合時の確認
1. uiController.getUIState() でUI状態確認
2. 対象DOM要素の存在確認
3. イベントリスナーの設定状況確認
```
