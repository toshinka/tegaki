# 🛠️ キャンバス等倍表示修正 ToDoリスト・作業手順書

## 📋 問題状況の分析

### 現状確認
- **設定値**: 400x400px
- **実際の表示**: 800x800px（2倍表示）
- **コード状態**: DPI関連設定は既に除去済み（resolution=1, autoDensity=false）
- **原因**: CSSまたはPixiJSの内部バッファサイズ処理で倍加発生

### 根本原因
1. **PixiJS内部処理**: `devicePixelRatio`が内部で適用されている可能性
2. **CSS倍加**: `image-rendering`やサイズ指定で予期しない拡大
3. **DOM操作**: キャンバス要素のサイズ設定で二重処理発生

### STEP 3: AppCore初期化処理の修正
**ファイル**: `js/app-core.js`
**目的**: PixiJS Application作成時のサイズ指定を明確化

#### 作業内容:
`initializePixiApp()` メソッド内を修正:
```javascript
// 🛠️ MODIFIED: PixiJS Application初期化（完全DPI無視版）
async initializePixiApp() {
    try {
        const canvasConfig = this.configManager.getCanvasConfig();
        const pixiConfig = this.configManager.getPixiConfig();
        
        // 🛠️ ADDED: DPI完全無視の強制設定
        const pixiOptions = {
            width: canvasConfig.width,
            height: canvasConfig.height,
            backgroundColor: canvasConfig.backgroundColor,
            antialias: pixiConfig.antialias,
            resolution: 1,
            autoDensity: false,
            // 🛠️ ADDED: DPI関連オプション明示的無効化
            useContextAlpha: true,
            powerPreference: 'default'
        };
        
        this.app = new PIXI.Application(pixiOptions);
        
        const canvasElement = document.getElementById('drawing-canvas');
        canvasElement.appendChild(this.app.view);
        
        // 🛠️ ADDED: キャンバス要素サイズ強制設定
        this.app.view.width = canvasConfig.width;
        this.app.view.height = canvasConfig.height;
        this.app.view.style.width = canvasConfig.width + 'px';
        this.app.view.style.height = canvasConfig.height + 'px';
        
        // 基本設定
        this.app.view.style.cursor = pixiConfig.cursor || 'crosshair';
        this.app.view.style.touchAction = 'none';
        
        console.log('✅ PixiJS完全等倍初期化完了');
        console.log(`📐 設定サイズ: ${canvasConfig.width}×${canvasConfig.height}px`);
        console.log(`📐 実際サイズ: ${this.app.view.width}×${this.app.view.height}px`);
        
    } catch (error) {
        console.error('❌ PixiJS等倍初期化失敗:', error);
        throw error;
    }
}
```

**参考メソッド**: `ConfigManager.getCanvasConfig()`, `PIXI.Application`
**検証**: コンソール出力で設定値と実際値の一致確認

### STEP 5: リサイズ処理の修正
**ファイル**: `js/app-core.js`
**目的**: リサイズ時もDPI倍加を防ぐ

#### 作業内容:
`resize()` メソッド内を修正:
```javascript
// 🛠️ MODIFIED: リサイズ処理（DPI完全無視版）
resize(newWidth, newHeight, centerContent = false) {
    if (!this.app) {
        console.warn('⚠️ PixiJSアプリが初期化されていません');
        return;
    }
    
    try {
        // 🛠️ MODIFIED: レンダラーリサイズ（DPI考慮なし）
        this.app.renderer.resize(newWidth, newHeight);
        
        // 🛠️ ADDED: キャンバス要素サイズ強制設定
        this.app.view.width = newWidth;
        this.app.view.height = newHeight;
        this.app.view.style.width = newWidth + 'px';
        this.app.view.style.height = newHeight + 'px';
        
        // プロパティ更新
        this.canvasWidth = newWidth;
        this.canvasHeight = newHeight;
        
        // ヒットエリア更新
        this.app.stage.hitArea = new PIXI.Rectangle(0, 0, newWidth, newHeight);
        
        console.log(`📐 等倍リサイズ完了: ${newWidth}×${newHeight}px`);
        
    } catch (error) {
        console.error('❌ 等倍リサイズエラー:', error);
    }
}
```

**参考メソッド**: `PIXI.Application.renderer.resize()`, `PIXI.Rectangle`
**検証**: リサイズ後のキャンバスサイズ確認

function verifyCanvasScaling() {

    if (!canvas) {
        console.error('❌ キャンバス要素が見つかりません');

        canvasWidth: canvas.width,
        canvasHeight: canvas.height,

        styleHeight: parseInt(canvas.style.height),
        // 計算値

        computedHeight: parseInt(getComputedStyle(canvas).height),
        // PixiJS設定

        // DPR
        devicePixelRatio: window.devicePixelRatio
    };
    
    console.table(results);
    
    // 判定
    const expectedWidth = 400;
    const expectedHeight = 400;
    const isCorrect = (
        results.canvasWidth === expectedWidth &&
        results.canvasHeight === expectedHeight &&
        results.styleWidth === expectedWidth &&
        results.styleHeight === expectedHeight

    if (isCorrect) {
        console.log('✅ 等倍表示成功！');

        console.log('❌ まだ倍加されています');

    return results;
}
