# 🔧 デバイスピクセル比対応 修正作業手順書

## 📋 問題分析

### 🚨 現在の問題
- キャンバスサイズが設定値より大きく表示される
- 高解像度ディスプレイ（Retina等）でのデバイスピクセル比考慮漏れ
- PixiJS初期化時の`resolution`と`autoDensity`オプション未設定

### ✅ GEMINI提案の妥当性確認
- **正確な原因特定**: `window.devicePixelRatio`の未考慮
- **適切な解決方法**: PixiJS Application初期化時のオプション追加
- **実装難易度**: 低（既存コード変更箇所が限定的）
- **副作用リスク**: 非常に低（標準的なPixiJS機能）

```javascript
// 解像度診断メソッド（canvas-manager.js に追加）
runResolutionDiagnosis() {
  console.group('🔍 解像度診断');
  
  const diagnosis = {
    device: {
      pixelRatio: window.devicePixelRatio,
      screenSize: {
        width: window.screen.width,
        height: window.screen.height
      },
      viewport: {
        width: window.innerWidth,
        height: window.innerHeight
      }
    },
    pixi: {
      resolution: this.app.renderer.resolution,
      autoDensity: this.app.renderer.plugins.interaction?.autoPreventDefault,
      canvasSize: {
        actual: {
          width: this.app.view.width,
          height: this.app.view.height
        },
        css: {
          width: this.app.view.style.width,
          height: this.app.view.style.height
        }
      }
    },
    expected: {
      width: this.width,
      height: this.height
    }
  };
  
  console.log('📊 診断結果:', diagnosis);
  
  // 問題検出
  const issues = [];
  
  if (!diagnosis.pixi.resolution || diagnosis.pixi.resolution === 1) {
    if (diagnosis.device.pixelRatio > 1) {
      issues.push('高解像度ディスプレイでresolution未設定');
    }
  }
  
  if (diagnosis.pixi.canvasSize.actual.width !== this.width * diagnosis.pixi.resolution) {
    issues.push('キャンバス実際サイズが期待値と不一致');
  }
  
  if (issues.length > 0) {
    console.warn('⚠️ 検出された問題:', issues);
  } else {
    console.log('✅ 解像度設定は正常です');
  }
  
  console.groupEnd();
  
  return diagnosis;
}
```

## 📝 実装コードサンプル

### 🎯 最終的な修正コード（完全版）
```javascript
// managers/canvas-manager.js の修正箇所

// デバイスピクセル比対応の安全な初期化
initializePixiApp(width, height) {
  try {
    // デバイス情報取得
    const devicePixelRatio = window.devicePixelRatio || 1;
    
    // ConfigManager連携（利用可能時）
    const pixiConfig = this.configManager?.getPixiConfig() || {};
    const resolution = pixiConfig.resolution || devicePixelRatio;
    const autoDensity = pixiConfig.autoDensity !== false;
    
    console.log(`🎨 PixiJS初期化: ${width}×${height}, resolution=${resolution}`);
    
    // PixiJS Application 初期化
    this.app = new PIXI.Application({
      width: width,
      height: height,
      backgroundColor: 0xffffff,
      antialias: true,
      resolution: resolution,           // ★ デバイスピクセル比対応
      autoDensity: autoDensity,         // ★ CSS自動調整
    });
    
    // 状態管理更新
    if (window.StateManager) {
      window.StateManager.updateSystemState('canvas', 'resolution', {
        devicePixelRatio: devicePixelRatio,
        pixiResolution: resolution,
        autoDensity: autoDensity,
        timestamp: Date.now()
      });
    }
    
    // コンテナサイズ調整
    this.updateContainerSize(width, height);
    
    console.log('✅ PixiJS初期化完了');
    
  } catch (error) {
    console.error('❌ PixiJS初期化エラー:', error);
    
    if (window.ErrorManager) {
      window.ErrorManager.showCriticalError(
        `キャンバス初期化に失敗: ${error.message}`
      );
    }
    
    throw error;
  }
}
```

この手順書に従って修正を行えば、デバイスピクセル比の問題が解決され、すべての解像度のディスプレイで適切なキャンバスサイズが表示されるようになります。
