🔍 検査結果
1. pixi-extensions.js

現在の役割は PixiJS 向けのユーティリティや拡張機能集。

ツールバーのロジックを直接持たせるのは責務が大きくなりすぎる。

アイコン取得の責務までは持たせるべきでなく、**Pixi に変換する共通処理（例：SVG → Texture）**を提供する形が妥当。

2. index.html

現状、node_modules 配下のライブラリを参照していない。

@tabler/icons-react は React コンポーネント形式で提供されているため、そのまま <script> import しても動かない。

そのため:

React 依存を外して SVG データだけを取得する。

もしくは、@tabler/icons パッケージを利用（React なし）。

👉 よって index.html に import 系の修正が必要。pixi-extensions.js だけを触っても完結しない。

3. app-core.js

アプリの初期化・PixiJS Application 管理が中心。

ここに「アイコン取得処理」までは書かない。

ただし、Sidebar UI コンポーネントを登録する責務はあるので、IconLoader 的なクラスを呼び出す形で修正。

4. main.js

起動スクリプト。UI 要素をまとめて登録する部分あり。

Sidebar のツールボタン生成処理に、アイコンロード → Sprite化 → ボタン登録の流れを追加する必要あり。

📝 改修計画書
目的

サイドバーのツールアイコンに @tabler/icons のSVGアイコンを node_modules/ から利用。

React依存の @tabler/icons-react は不適切 → @tabler/iconsに切り替える。

fetch API 分割構造を維持。

改修ステップ
ステップ 1: パッケージ準備

@tabler/icons-react → ❌（React依存のため非推奨）

@tabler/icons → ✅（SVGデータを直接参照可能）

npm install @tabler/icons

ステップ 2: index.html 修正

node_modules/@tabler/iconsのアイコンを取得できるようにする。

ESM import 可能な形に修正:

<script type="module">
  import { IconPencil, IconEraser } from "../node_modules/@tabler/icons/index.js";

  // グローバルに登録
  window.TABLER_ICONS = { IconPencil, IconEraser };
</script>

ステップ 3: pixi-extensions.js 修正

SVG を PixiJS 用 Texture に変換する関数を追加。

例：

export async function svgToTexture(svgContent, pixiApp) {
  const svgBlob = new Blob([svgContent], { type: "image/svg+xml;charset=utf-8" });
  const url = URL.createObjectURL(svgBlob);
  const texture = await PIXI.Assets.load(url);
  URL.revokeObjectURL(url);
  return texture;
}


👉 pixi-extensions.js の責務は「Pixiに変換する共通関数」まで。

ステップ 4: app-core.js 修正

Sidebar 管理クラスに loadIcon メソッドを追加し、svgToTexture を利用。

import { svgToTexture } from "../libs/pixi-extensions.js";

class Sidebar {
  constructor(app) {
    this.app = app;
    this.container = new PIXI.Container();
    app.stage.addChild(this.container);
  }

  async addToolButton(iconSvg, onClick) {
    const texture = await svgToTexture(iconSvg, this.app);
    const sprite = new PIXI.Sprite(texture);
    sprite.interactive = true;
    sprite.on("pointertap", onClick);
    this.container.addChild(sprite);
  }
}

ステップ 5: main.js 修正

起動時に Tabler アイコンを取得して Sidebar に登録。

import { Sidebar } from "./app-core.js";

const sidebar = new Sidebar(app);

sidebar.addToolButton(TABLER_ICONS.IconPencil, () => {
  console.log("Pencil selected");
});

sidebar.addToolButton(TABLER_ICONS.IconEraser, () => {
  console.log("Eraser selected");
});

✅ 結論

index.html の修正は必須（@tabler/icons-react は直接使えない → SVG版をimportする必要あり）。

pixi-extensions.js だけでは不十分、責務ごとに以下に分けるべき:

index.html → Tabler SVG import

pixi-extensions.js → SVG → Pixi変換処理

app-core.js → Sidebarクラスでボタン生成

main.js → 起動時に Sidebar にツール登録
