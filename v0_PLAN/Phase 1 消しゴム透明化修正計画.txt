# Phase 1 消しゴム透明化修正計画 v2.0 - マスク方式実装

## 🚨 現在の問題・検証結果

### 現象
- 消しゴムが「背景色で上書き」している挙動
- レイヤーのピクセルのアルファ（透明度）を消していない  
- Layer2からLayer1を消しているように見えてしまう

### 原因調査結果
- ❌ **PIXI.BLEND_MODES.ERASE存在しない**: [PixiJS公式ドキュメント](https://api.pixijs.io/@pixi/constants/PIXI/BLEND_MODES.html)によると、WebGLレンダラーではNORMAL, ADD, MULTIPLY, SCREENのみサポート
- ❌ **元計画の問題点**: 存在しないAPIに依存した設計
- 🔍 **背景色塗り問題**: 現在の実装は背景色 (`0xf0e0d6`) で塗ることで見かけ上消している

### 技術的制約
- 実行環境：Chrome 最新、ローカル HTML 直開き (`file://`)
- PixiJS v8.13.0：ERASEブレンドモード未対応確定
- 禁止事項：Canvas2D, TypeScript, モジュール化, bundler不使用

## 🎯 Phase 1 新修正方針：マスク方式

### アプローチ変更
1. **マスク方式による透明化**: `PIXI.Graphics.mask` プロパティを活用
2. **レイヤー別マスク管理**: 各レイヤーに専用マスクGraphicsを設定
3. **反転マスク作成**: 消しゴム領域を除外するマスクを動的生成
4. **非破壊編集**: 元の描画データを保持、マスクのみ更新

### 利点
- ✅ PixiJS v8標準機能のみ使用（互換性100%）
- ✅ 真の透明化実現（背景色塗りではない）
- ✅ レイヤー分離維持
- ✅ 非破壊編集対応（undo/redo準備）
- ✅ メモリ効率的

### Step 1: layer-tool-ui.js - LayerManager拡張

#### 1.1 レイヤー構造拡張（マスク対応）
```javascript
createLayer(name = null) {
    const layerId = this.nextLayerId++;
    const layerName = name || `レイヤー${layerId}`;
    
    // コンテナ作成
    const container = new PIXI.Container();
    container.name = layerName;
    container.visible = true;
    
    // 🆕 マスク用Graphics作成
    const maskGraphics = new PIXI.Graphics();
    const canvasState = MainController.getState('canvas');
    
    // 初期マスク：全体表示（白塗り）
    maskGraphics.rect(0, 0, canvasState.width, canvasState.height);
    maskGraphics.fill(0xFFFFFF);
    
    // 🆕 マスクを適用
    container.mask = maskGraphics;
    this.engine.containers.world.addChild(maskGraphics); // マスクもworldに追加
    
    const layer = {
        id: layerId,
        name: layerName,
        container: container,
        maskGraphics: maskGraphics,  // 🆕 マスクGraphics保存
        erasedAreas: [],             // 🆕 消去領域記録（undo/redo用）
        visible: true,
        paths: [],
        isBackground: false
    };
    
    this.layers.set(layerId, layer);
    this.engine.containers.world.addChild(container);
    
    return layer;
}
```

```javascript
createBackgroundLayer() {
    const layerId = 0;
    const layerName = '背景';
    
    const container = new PIXI.Container();
    container.name = layerName;
    container.visible = true;
    
    // 背景色作成
    const backgroundGraphics = new PIXI.Graphics();
    backgroundGraphics.rect(0, 0, MainController.getState('canvas').width, MainController.getState('canvas').height);
    backgroundGraphics.fill(0xf0e0d6);
    container.addChild(backgroundGraphics);
    
    // 🚨 背景レイヤーにはマスクを設定しない
    const layer = {
        id: layerId,
        name: layerName,
        container: container,
        maskGraphics: null,          // 背景レイヤーはマスクなし
        erasedAreas: [],
        visible: true,
        paths: [],
        isBackground: true,
        backgroundGraphics: backgroundGraphics
    };
    
    this.layers.set(layerId, layer);
    this.engine.containers.world.addChild(container);
    this.activeLayerId = layerId;
    
    // レイヤー1作成
    this.createLayer('レイヤー1');
    this.setActiveLayer(1);
}
```

### Step 2: layer-tool-ui.js - DrawingEngine拡張（消しゴム対応）

#### 2.1 消しゴムツール検出と処理分岐
```javascript
createPath(worldX, worldY, settings) {
    const path = {
        id: `path_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`,
        graphics: new PIXI.Graphics(),
        points: [],
        tool: settings.tool,    // 🆕 ツール種別保存
        color: settings.color,
        size: settings.size,
        opacity: settings.opacity,
        isComplete: false,
        layerId: MainController.getState('activeLayerId')
    };
    
    if (settings.tool === 'eraser') {
        // 🆕 消しゴム処理：マスクに穴を開ける
        this.eraseAtPoint(worldX, worldY, settings.size);
        
        // 消しゴムパスは視覚的には作成しない（マスク更新のみ）
        return path; // 空のパス返却
    } else {
        // 通常ペン処理
        path.graphics.circle(worldX, worldY, settings.size / 2);
        path.graphics.fill({ color: settings.color, alpha: settings.opacity });
    }
    
    path.points.push({ x: worldX, y: worldY, size: settings.size });
    
    window.LayerManager?.addPathToActiveLayer(path);
    this.paths.push(path);
    this.currentPath = path;
    
    return path;
}
```

#### 2.2 マスク穴開け処理
```javascript
eraseAtPoint(worldX, worldY, size) {
    const activeLayer = window.LayerManager?.getActiveLayer();
    if (!activeLayer || !activeLayer.maskGraphics || activeLayer.isBackground) {
        return; // 背景レイヤーや無効レイヤーでは消去無効
    }
    
    // 🆕 マスクに黒い円を描画（透明化領域）
    activeLayer.maskGraphics.circle(worldX, worldY, size / 2);
    activeLayer.maskGraphics.fill(0x000000, 1.0); // 黒 = マスク除外
    
    // 🆕 消去記録保存（undo/redo準備）
    activeLayer.erasedAreas.push({
        x: worldX,
        y: worldY,
        size: size,
        timestamp: Date.now()
    });
    
    MainController.recordAction('ERASE_POINT', {
        layerId: activeLayer.id,
        x: worldX,
        y: worldY,
        size: size
    });
}

extendErase(worldX, worldY, size) {
    const activeLayer = window.LayerManager?.getActiveLayer();
    if (!activeLayer || !activeLayer.maskGraphics || activeLayer.isBackground) {
        return;
    }
    
    // 消去領域を線で接続
    const lastErased = activeLayer.erasedAreas[activeLayer.erasedAreas.length - 1];
    if (lastErased) {
        // ライン消去（円の連続描画）
        const distance = Math.sqrt((worldX - lastErased.x) ** 2 + (worldY - lastErased.y) ** 2);
        const steps = Math.max(1, Math.ceil(distance / (size / 3)));
        
        for (let i = 1; i <= steps; i++) {
            const t = i / steps;
            const px = lastErased.x + (worldX - lastErased.x) * t;
            const py = lastErased.y + (worldY - lastErased.y) * t;
            
            activeLayer.maskGraphics.circle(px, py, size / 2);
            activeLayer.maskGraphics.fill(0x000000, 1.0);
        }
    }
    
    // 現在位置記録
    activeLayer.erasedAreas.push({
        x: worldX,
        y: worldY,
        size: size,
        timestamp: Date.now()
    });
}
```

#### 2.3 ツール別処理分岐強化
```javascript
handleDrawStartRequest(payload) {
    const activeLayer = window.LayerManager?.getActiveLayer();
    if (!activeLayer || activeLayer.id !== payload.layerId) {
        return;
    }
    
    const toolSettings = window.ToolManager?.getCurrentToolSettings();
    if (!toolSettings) return;
    
    if (toolSettings.tool === 'eraser') {
        // 🆕 消しゴムモード
        this.eraseAtPoint(payload.worldX, payload.worldY, toolSettings.size);
        this.currentErase = { active: true, size: toolSettings.size }; // 消去状態追跡
    } else {
        // 通常描画モード  
        const settings = {
            tool: toolSettings.tool,
            color: toolSettings.color,
            size: toolSettings.size,
            opacity: toolSettings.opacity
        };
        
        this.createPath(payload.worldX, payload.worldY, settings);
    }
}

handleDrawContinueRequest(payload) {
    if (this.currentErase && this.currentErase.active) {
        // 🆕 消しゴム延長
        this.extendErase(payload.worldX, payload.worldY, this.currentErase.size);
    } else if (this.currentPath) {
        // 通常パス延長
        const activeLayer = window.LayerManager?.getActiveLayer();
        if (activeLayer && this.currentPath.layerId === activeLayer.id) {
            this.extendPath(this.currentPath, payload.worldX, payload.worldY);
        }
    }
}

handleDrawEndRequest(payload) {
    if (this.currentErase && this.currentErase.active) {
        // 🆕 消しゴム終了
        this.currentErase = { active: false };
        
        MainController.recordAction('ERASE_COMPLETE', {
            layerId: MainController.getState('activeLayerId'),
            pointCount: window.LayerManager?.getActiveLayer()?.erasedAreas.length || 0
        });
    } else if (this.currentPath) {
        // 通常パス終了
        const activeLayer = window.LayerManager?.getActiveLayer();
        if (activeLayer && this.currentPath.layerId === activeLayer.id) {
            this.finalizePath(this.currentPath);
        }
    }
}
```

### Step 3: layer-tool-ui.js - リサイズ対応

#### 3.1 マスクサイズ更新
```javascript
resize(newWidth, newHeight) {
    MainController.setState('canvas', { width: newWidth, height: newWidth });
    
    this.app.renderer.resize(newWidth, newHeight);
    
    // カメラマスク更新
    if (this.containers.camera.mask) {
        this.containers.camera.mask.clear();
        this.containers.camera.mask.rect(0, 0, newWidth, newHeight);
        this.containers.camera.mask.fill(0x000000);
    }
    
    this.containers.camera.hitArea = new PIXI.Rectangle(0, 0, newWidth, newHeight);
    
    // 🆕 全レイヤーのマスクサイズ更新
    const layers = window.LayerManager?.getAllLayers() || [];
    layers.forEach(layer => {
        if (layer.maskGraphics) {
            // 既存マスクの状態を保持しつつサイズ拡張
            const tempCanvas = document.createElement('canvas');
            const tempCtx = tempCanvas.getContext('2d');
            tempCanvas.width = newWidth;
            tempCanvas.height = newHeight;
            
            // 白で初期化（表示領域）
            tempCtx.fillStyle = '#FFFFFF';
            tempCtx.fillRect(0, 0, newWidth, newHeight);
            
            // 既存の消去領域を再描画
            tempCtx.fillStyle = '#000000';
            layer.erasedAreas.forEach(area => {
                tempCtx.beginPath();
                tempCtx.arc(area.x, area.y, area.size / 2, 0, Math.PI * 2);
                tempCtx.fill();
            });
            
            // マスクをクリアして再構築
            layer.maskGraphics.clear();
            
            // Canvas to PIXI Graphics変換（簡略化）
            layer.maskGraphics.rect(0, 0, newWidth, newHeight);
            layer.maskGraphics.fill(0xFFFFFF);
            
            // 消去領域再適用
            layer.erasedAreas.forEach(area => {
                layer.maskGraphics.circle(area.x, area.y, area.size / 2);
                layer.maskGraphics.fill(0x000000);
            });
        }
    });
    
    // 背景レイヤーサイズ更新
    window.LayerManager?.updateBackgroundSize(newWidth, newHeight);
    
    MainController.emit('ui-canvas-resize', { width: newWidth, height: newHeight });
}
```

```javascript
// error-service.js 追加
checkPixiMaskCompatibility() {
    const issues = [];
    
    if (!window.PIXI) {
        issues.push('PixiJS not loaded');
        return issues;
    }
    
    if (!PIXI.Graphics) {
        issues.push('PIXI.Graphics not available');
    }
    
    // マスク機能テスト
    try {
        const testContainer = new PIXI.Container();
        const testMask = new PIXI.Graphics();
        testMask.rect(0, 0, 100, 100);
        testMask.fill(0xFFFFFF);
        testContainer.mask = testMask; // マスク設定テスト
        
        if (testContainer.mask !== testMask) {
            issues.push('PIXI mask functionality not working');
        }
        
        // クリーンアップ
        testContainer.destroy();
        testMask.destroy();
    } catch (error) {
        issues.push(`PIXI mask test failed: ${error.message}`);
    }
    
    if (issues.length > 0) {
        this.reportError('MASK_001', {
            message: 'Pixi mask compatibility issues detected',
            issues: issues,
            version: PIXI.VERSION || 'unknown'
        });
        
        return issues;
    }
    
    console.log('✅ PixiJS Mask compatibility OK');
    return [];
}
```

## 🔧 使用方法・挙動

### 消しゴムの動作
1. **初期状態**: レイヤーマスクは全体白（全表示）
2. **消しゴム使用**: マスクに黒円描画 → 該当部分が透明化
3. **レイヤー重ね**: 下のレイヤーが透けて見える
4. **背景レイヤー**: マスクなし、消しゴム無効

### レイヤー切り替え
- Layer1で消去 → Layer1のマスクのみ更新
- Layer2に切り替え → Layer2は影響なし
- 真のレイヤー分離実現
