## 🛠️ 緊急修正手順（段階別）

### STEP 1: ToolManager構文エラー緊急修復（最優先）

#### 原因分析
- `tool-manager.js:9` で `Invalid or unexpected token`
- 可能性：BOM文字混入、不可視制御文字、全角記号、未閉じテンプレートリテラル、ESM構文問題

#### 修正方針
1. **文字コード統一**: UTF-8 (BOM無し)
2. **構文検証**: ESM export/import整理
3. **最小動作骨格**: 必須API実装
4. **Manager統一ライフサイクル**: configure/attach/init/isReady/dispose

#### 必須実装API（ToolManager）
```javascript
class ToolManager {
  // ライフサイクル（必須）
  configure(config) → void
  attach(context) → void  
  init() → Promise<void>
  isReady() → boolean
  dispose() → void
  
  // Manager連携（必須）
  setManagers(managers) → void
  verifyInjection() → boolean
  
  // ツール管理（必須）
  initializeV8Tools() → Promise<void>
  registerTool(name, tool) → boolean
  switchTool(name) → boolean
  getCurrentTool() → Tool|null
}
```
