# Manager注入エラー修正計画書 v8対応版

## 🔍 問題分析

### 現状のエラー
```
pen: Missing required managers: canvas, coordinate, record
```

### 問題の原因
1. **Manager登録キー不一致**: AppCoreで作成したManagerとAbstractToolが要求するキーが不一致
2. **Manager注入タイミング問題**: ToolManager作成時にManagerがまだ注入されていない
3. **Manager参照方式の不統一**: Mapとobjectの混在使用

## 📋 現在のManager登録状況

### AppCore登録キー (app-core.js:369)
```javascript
managers.set('coordinate', coordinateManager);
managers.set('record', recordManager);  
managers.set('config', configManager);
managers.set('error', errorManager);
managers.set('event', eventBus);
managers.set('shortcut', shortcutManager);
managers.set('navigation', navigationManager);
managers.set('canvas', canvasManager); // ← この順序
```

### AbstractTool要求キー (abstract-tool.js:141)
```javascript
const requiredManagers = ['canvas', 'coordinate', 'record'];
```

## 🎯 修正方針

### 1. Manager初期化順序保証
```
1. CanvasManager → 最初に作成（描画基盤）
2. その他Manager → 順次作成
3. ToolManager → Manager群注入後に作成
4. Tool群 → 依存Manager確認後に初期化
```

### 2. Manager注入の統一化
```javascript
// AppCore → ToolManager
toolManager.setManagers(this.managers); // Map<string, Manager>

// ToolManager → AbstractTool  
tool.setManagers(managersObject); // object形式に変換
```

### 3. 依存関係チェック強化
```javascript
// 必須Manager存在確認
const requiredKeys = ['canvas', 'coordinate', 'record'];
const availableKeys = Object.keys(managers);
const missing = requiredKeys.filter(key => !availableKeys.includes(key));
```
