❌ 主なエラー原因

tool-manager.js:132 Uncaught SyntaxError: await is only valid in async functions

await が通常関数内で使われている（async 修飾子不足）。

Pure JS (ESM禁止) なので、トップレベル await も不可。

canvas-manager.js:258 Uncaught SyntaxError: Unexpected token '{'

構文エラー。たぶんオブジェクト展開や import 風記法が混じっている。

ESM記法 (import { ... }) や export { ... } が紛れ込んでいる可能性。

app-core.js / main.js

CanvasManager / ToolManager の初期化時に CoordinateManagerを正しく注入できていない。

メッセージ：

⚠️ CanvasManager利用不可
⚠️ ToolManager未初期化
⚠️ BoundaryManager: CoordinateManager未提供


原因：CoordinateManager がグローバルに公開されていない or Manager間の依存渡し忘れ。

boundary-manager.js

canvasElement未提供 → CanvasManager の初期化不全に連鎖。

✅ 依存関係の整理

各マネージャの依存性をまとめると：

CoordinateManager

統一座標系の中心。全ての座標計算をここに集約。

グローバル参照：window.CoordinateManager

CanvasManager

初期化時に CoordinateManager を受け取る必要あり。

推奨：CanvasManager.initialize(coordinateManager, canvasElement)

ToolManager

座標関連ツールで CoordinateManager を利用。

推奨：ToolManager.initialize(coordinateManager)

BoundaryManager

Canvasの拡張ヒットエリア計算で CoordinateManager 必須。

CanvasManager が正しく初期化されれば自動連携。

coordinates.js

すべて削除予定。現在は警告を出すだけ。

🛠 対策案
1. CoordinateManagerをグローバル公開
// coordinate-manager.js
(function(global) {
  function CoordinateManager() {
    // 座標関連の処理
  }

  CoordinateManager.prototype.calculateDistance = function(p1, p2) { ... };
  CoordinateManager.prototype.calculateAngle = function(p1, p2) { ... };

  // グローバル公開
  global.CoordinateManager = CoordinateManager;
})(window);

2. app-core.js: Manager統合時に注入
function initializeManagers() {
  try {
    // ① 座標管理の統合
    window.coordinateManager = new CoordinateManager();

    // ② CanvasManager
    window.canvasManager = new CanvasManager();
    canvasManager.initialize(window.coordinateManager, document.getElementById("canvas"));

    // ③ ToolManager
    window.toolManager = new ToolManager();
    toolManager.initialize(window.coordinateManager);

    // ④ BoundaryManager
    window.boundaryManager = new BoundaryManager();
    boundaryManager.initialize(canvasManager.getCanvasElement(), window.coordinateManager);

    console.log("✅ Managers initialized with CoordinateManager");
  } catch (err) {
    console.error("Manager init failed:", err);
  }
}

3. tool-manager.js 修正
// NG: function 内で await
// OK: async function にする
ToolManager.prototype.initialize = async function(coordinateManager) {
  this.coordinateManager = coordinateManager;
  // await 使用可
};

4. canvas-manager.js 構文エラー修正

Unexpected token '{' の場所を確認し、ESM風記法を排除。

Pure JS形式に統一：

function CanvasManager() { ... }
CanvasManager.prototype.initialize = function(coordinateManager, canvasElement) { ... };
window.CanvasManager = CanvasManager;
