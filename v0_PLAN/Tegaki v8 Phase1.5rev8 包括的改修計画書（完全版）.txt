## 🔍 架空メソッド・参照エラー検出

### 1. ToolManager の verifyInjection() 問題
- **問題**: DrawContainer 参照時に verifyInjection() が必要だがスキップされている
- **原因**: 初期化フローで verifyInjection() が呼ばれていない

### 2. CanvasManager との連携問題
- **問題**: CanvasManager は DrawContainer を作成済みだが、ToolManager から参照不可
- **原因**: Manager間の参照注入タイミングの問題

### 3. 二重初期化による状態不整合
- **問題**: 同一 Manager が複数回初期化される
- **原因**: Bootstrap/main.js の重複実行

## 🛠️ 具体的修正計画

### STEP 1: 緊急エラー修正（最優先）

#### 1.1 ToolManager.js 修正
**問題**: verifyInjection() の呼び出し不足
```javascript
// 現在のエラー原因
async initializeV8Tools() {
    // verifyInjection() が呼ばれていない
    const drawContainer = this.canvasManager.getDrawContainer(); // ← エラー
}

// 修正版
async initializeV8Tools() {
    // まず検証を実行
    if (!this.verifyInjection()) {
        throw new Error('Manager injection verification failed');
    }
    const drawContainer = this.canvasManager.getDrawContainer();
}
```

#### 1.2 AppCore.js 修正
**問題**: ToolManager 初期化時の verifyInjection() 呼び出し不足
```javascript
// 修正版
async initializeToolManager() {
    await this.managers.toolManager.configure(this.config);
    await this.managers.toolManager.attach(this.context);
    await this.managers.toolManager.init();
    
    // Manager注入
    this.managers.toolManager.setManagers(this.managers);
    
    // ★追加: 注入検証
    if (!this.managers.toolManager.verifyInjection()) {
        throw new Error('ToolManager injection verification failed');
    }
    
    // ツール初期化
    await this.managers.toolManager.initializeV8Tools();
}
```

### STEP 3: Manager 間連携強化

#### 3.1 CanvasManager.js 修正
**問題**: DrawContainer の確実な提供
```javascript
getDrawContainer() {
    if (!this._status.ready) {
        throw new Error('CanvasManager not ready - call init() first');
    }
    if (!this.drawContainer) {
        throw new Error('DrawContainer not created');
    }
    return this.drawContainer;
}

// デバッグ用状態確認メソッド追加
getContainerStatus() {
    return {
        ready: this._status.ready,
        hasStage: !!this.stage,
        hasWorldContainer: !!this.worldContainer,
        hasDrawContainer: !!this.drawContainer,
        appReady: !!this.app
    };
}
```
