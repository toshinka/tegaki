## 🎯 **Dictionary v8の役割変更**

### 📄 従来（v7）vs v8新方式
```
【v7方式】集中管理
Symbol Dictionary = 全情報の一次保管場所
├── 全メソッド定義
├── 全依存関係  
├── 全Manager詳細
└── 手動更新（更新遅延リスク）

【v8方式】分散ヘッダー集約
各ファイルヘッダー = 一次情報（@provides/@uses/@initflow等）
Symbol Dictionary = 集約・検索・整合性検証
├── ファイル横断検索
├── 依存関係可視化
├── 整合性自動検証
└── 用語統一保証
```

### 🔍 Dictionary新機能
1. **ファイル横断検索**: 「どのファイルがgetDrawContainer()を提供？」
2. **依存関係可視化**: Manager間の依存マトリックス表示
3. **整合性自動検証**: 呼び出し先メソッド存在確認
4. **用語統一保証**: API名・データ構造・定数の統一

### 🎨 TPFStroke（TegakiPathFormat v0.2）
```javascript
/**
 * 🎯 非破壊ストローク記録・完全可逆編集
 */
interface TPFStroke {
    id: string;              // UUID・重複禁止・必須
    kind: 'stroke' | 'erase'; // 操作種別・厳密enum
    layer: string;           // レイヤー名・Phase1.5='default'
    color: string;           // '#rrggbb'形式・厳守
    width: number;           // px単位・正数のみ
    opacity: number;         // 0.0-1.0・厳密範囲
    points: Point[];         // 座標配列・必須・空配列禁止
    path?: string;           // SVG 'd'属性・Phase1.5任意
    bounds?: Rectangle;      // 最適化用・自動計算
    meta: TPFMetadata;       // メタ情報・必須
}

interface Point {
    x: number;              // キャンバス座標X・必須
    y: number;              // キャンバス座標Y・必須
    pressure?: number;      // 0.0-1.0・Phase1.5任意
    time?: number;          // タイムスタンプ・Phase1.5任意
}

interface TPFMetadata {
    created: number;        // 作成タイムスタンプ・必須
    tool: 'pen' | 'eraser'; // 使用ツール・必須
    engine: 'pixi-v8';     // エンジン識別・v8固定
    version: '0.2';        // TPFバージョン・必須
}

interface Rectangle {
    x: number;             // 左上X座標
    y: number;             // 左上Y座標  
    width: number;         // 幅・正数
    height: number;        // 高さ・正数
}
```
