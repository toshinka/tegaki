# 🎨 シンボル辞典 v11 - 責務分離・統一定義版

## 🔄 更新概要（v10 → v11）

### 📋 責務分離対応
- **責務別API分類**: CanvasManager（レイヤー管理）、Tool（描画実装）、Manager（統合制御）
- **統一インターフェース定義**: AbstractTool基底クラス・共通メソッド規約
- **禁止・許可事項明確化**: 各コンポーネントの責務境界明確定義

### 🚨 重要な分類変更
```javascript
🔴 CanvasManager系API:
- レイヤー・Graphics配置専門API群
- 描画処理API削除・Tool委譲へ変更

🖊️ Tool系API:
- 描画主導権保持・座標制御専門API群
- AbstractTool統一インターフェース導入

🎯 Manager系API:
- 統合制御・配信管理専門API群
- 直接描画禁止・Tool委譲必須化
```

## 📊 責務分離APIカテゴリ

### 🔴 CanvasManager系API（レイヤー・ステージ管理専門）

#### 📋 許可されるAPI群
```javascript
// レイヤー管理（主責務）
addGraphicsToLayer(graphics, layerId)       // Graphics配置
removeGraphicsFromLayer(graphics, layerId)  // Graphics削除  
createNewLayer(layerId, type, options)       // レイヤー作成
setLayerVisibility(layerId, visible)         // 表示制御
getLayerForTool(toolName)                   // ツール用レイヤー取得

// PixiJS基盤管理

setupStage()                                // Stage設定
setupRenderer()                             // Renderer設定
resize(width, height, centerContent)        // リサイズ制御

// レンダリング制御
render()                                    // レンダリング実行
clear()                                     // 全体クリア
export(format, options)                     // 出力・保存

// Tool統合インターフェース
delegateToActiveTool(method, event)         // Tool委譲
attachToolGraphics(tool, graphics)          // Tool-Graphics接続
```

#### 🚫 禁止されるAPI群
```javascript
// 禁止：直接描画処理
❌ startDrawing()                           // Tool委譲必須
❌ continueDrawing()                        // Tool委譲必須  
❌ moveTo() / lineTo()                      // Tool責務
❌ Graphics生成・直接操作                    // Tool責務

// 禁止：座標変換
❌ screenToCanvas()                         // CoordinateManager責務
❌ extractCoordinates()                     // Tool + CoordinateManager責務
```

## 🏗️ システム統合パターン（責務分離準拠）

### 🎨 描画フロー統合パターン
```javascript
// 理想的な描画フロー（責務分離準拠）
User Input 
↓
CanvasManager.handlePointerEvent(event)    // イベント受信
↓ 
ToolManager.delegateToActiveTool(method, event) // Tool委譲
↓
PenTool.onPointerDown(event, canvasManager, coordinateManager) // Tool処理
↓
CoordinateManager.extractPointerCoordinates(event, rect) // 座標変換
↓
PenTool.startDrawing(validatedCoords)      // 描画実行・Graphics生成
↓
CanvasManager.addGraphicsToLayer(graphics, layerId) // レイヤー配置
↓
StateManager.updateComponentState('penTool', 'drawing', data) // 状態更新
↓
EventBus.safeEmit('drawing.started', data) // イベント通知
```

### 🔄 Tool登録・統合パターン
```javascript
// Tool登録時の統合手順
ToolManager.registerTool(toolName, toolInstance)
↓
CanvasManager.createToolGraphics(toolName)  // Tool用Graphics作成
↓
toolInstance.attachGraphics(graphics)      // Graphics接続
↓
CanvasManager.addGraphicsToLayer(graphics, layerId) // レイヤー配置
↓
StateManager.updateSystemState('toolManager', 'registered', data)
```

### 🏥 エラー処理統合パターン
```javascript
// 統合エラー処理フロー
Tool/Manager内でエラー発生
↓
ErrorManager.showError(type, message, options) // エラー分類・表示
↓
StateManager.updateSystemState('errorManager', 'error', errorData) // 状態記録
↓
EventBus.safeEmit('system.error', errorData) // エラー通知（連鎖防止）
↓
診断システムによる問題特定・解決支援
```
