## 🎯 新規問題登録: キャンバス外描画境界問題

### 問題概要
```javascript
🚨 キャンバス外描画境界問題:
問題: キャンバス境界外からマウスを引いてキャンバス内に入っても描画開始されない
影響: エッジ付近での描画作業の利便性大幅低下
要求水準: Clip Studio Paint・Photoshop水準の境界越え対応
緊急度: 中（利便性向上・ユーザビリティ問題）

具体的症状:
1. キャンバス外でmousedown → キャンバス内mousemove → 描画なし
2. キャンバス境界1pxでの描画中断（厳密すぎる境界）
3. PointerEvent対応不完全（タブレット・タッチ対応不十分）
4. 描画継続性問題（境界をまたぐ線の途切れ）
```

### 影響範囲分析
```javascript
影響コンポーネント:
📝 pen-tool.js: startDrawing()・continueDrawing()境界判定
📝 eraser-tool.js: 同様の境界判定問題
📝 canvas-manager.js: キャンバス領域・イベント処理
📝 tool-manager.js: ツールイベント統合・境界管理
📝 app-core.js: PixiJSイベント・ヒットエリア設定

技術課題:
1. PixiJS hitAreaによる厳密な境界制限
2. mousedown境界外→mousemove境界内の未対応
3. PointerEvent・TouchEventの境界越え処理
4. 描画継続性のロジック改修必要
```
