### **Phase 2: アンドゥ問題修正（20分）**

#### **Task 2.1**: Sprite nullエラー修正（10分）
- **対象ファイル**: `history-manager.js`, `app-core.js`
- **具体的修正**:
  ```javascript
  // history-manager.js
  restoreDrawingState(state) {
    if (!state || !state.texture) {
      console.warn('Invalid state for restore');
      return;
    }
    
    // nullチェック強化
    if (!state.texture.baseTexture || state.texture.baseTexture.destroyed) {
      console.warn('Invalid texture for restore');
      return this.createEmptySprite();
    }
    
    const sprite = new PIXI.Sprite(state.texture);
    return sprite;
  }
  
  createEmptySprite() {
    return new PIXI.Sprite(PIXI.Texture.EMPTY);
  }
  ```

#### **Task 2.2**: 画像劣化防止（10分）
- **対象ファイル**: `history-manager.js`
- **具体的修正**:
  ```javascript
  captureDrawingState() {
    // 高解像度キャプチャ
    const originalResolution = this.app.renderer.resolution;
    this.app.renderer.resolution = 2; // 2倍解像度
    
    const renderTexture = PIXI.RenderTexture.create({
      width: this.app.screen.width,
      height: this.app.screen.height,
      resolution: 2
    });
    
    this.app.renderer.render(this.app.stage, renderTexture);
    
    // 元の解像度に戻す
    this.app.renderer.resolution = originalResolution;
    
    return {
      texture: renderTexture,
      timestamp: Date.now()
    };
  }
  ```
