# ふたば☆ちゃんねるお絵かきツール - 問題修正計画書

## 📋 発生している問題の分析

### 🚨 **問題1**: ペンツールでアンドゥ後再描画時のPixiJSエラー
```
ApplicationError: Script error.
Sprite.ts:443 Uncaught TypeError: Cannot read properties of null (reading 'width')
```

**原因分析**:
- PixiJS Spriteオブジェクトが null/undefined になっている
- アンドゥ処理後の描画復元時にGraphicsオブジェクトの参照が不正
- 履歴管理とPixiJS描画オブジェクト間の同期不整合

### 🚨 **問題2**: 消しゴムツールへの切り替えが不可能
**原因分析**:
- `tool-manager.js`でツール切り替えは正常動作
- `ui-manager.js`の`handleGeneralToolButtonClick`でeraser-toolは処理されている
- しかし実際のツール状態変更が反映されていない可能性

### 🚨 **問題3**: ペンツールポップアップが表示されない
**原因分析**:
- `ui-manager.js`からペンツール専用機能が`PenToolUI`に移譲済み
- `popup-manager.js`はPenToolUI配下で動作するが、初期化・連携に問題
- ペンツールボタンクリック処理の重複・競合
