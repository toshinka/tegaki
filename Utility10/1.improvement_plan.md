# ペンツールポップアップUI改修計画書

## 📋 改修概要
ペンアイコンをクリックした時に表示されるポップアップ（`.pen-settings`）のUIを改善する。

## 🎯 改修要求事項

### 1. ペンサイズプリセットの配置改善
**現状の問題:**
- プリセットアイテムが右に寄っている
- アクティブ時の○枠が太くなる効果が不要

**改修内容:**
- プリセットアイテムを均等配置にする
- アクティブ時の○枠太線効果を削除し、背景色の変更のみとする

### 2. スライダー数値表示の修正
**現状の問題:**
- スライダー上の数値が右上にズレている
- プリセット数値は動くが、スライダー上の数値が更新されない

**改修内容:**
- 数値表示位置を適切な場所に修正
- スライダー値変更時の数値表示を正しく反映

### 3. ポップアップサイズの最適化
**現状の問題:**
- ブラウザのタブバー・アドレスバーがある状態で下部が切れる
- 全体的にやや大きすぎる

**改修内容:**
- ポップアップ全体をひと回り小さくする
- 要素間の隙間を詰める

## 🔧 技術的改修ポイント

### A. styles.css の修正箇所

#### 1. プリセット配置の改善
```css
/* 修正対象: .size-presets */
.size-presets {
    justify-content: space-between; /* → space-around または均等配置 */
    gap: 6px; /* → 調整 */
}

/* 修正対象: .size-preset-item.active .size-preview-frame */
.size-preset-item.active .size-preview-frame {
    border: 3px solid var(--futaba-maroon); /* → 削除 */
}
```

#### 2. スライダー数値表示の修正
```css
/* 修正対象: .slider-value */
.slider-value {
    position: absolute;
    right: 0;           /* → 位置調整 */
    top: -20px;         /* → 位置調整 */
}
```

#### 3. ポップアップサイズの最適化
```css
/* 修正対象: .pen-settings */
.pen-settings {
    width: 320px;       /* → 280px程度に縮小 */
    min-height: 400px;  /* → 350px程度に縮小 */
}

/* 修正対象: .popup-panel */
.popup-panel {
    padding: 20px;      /* → 16px程度に縮小 */
}

/* 修正対象: .setting-group */
.setting-group {
    margin-bottom: 16px; /* → 12px程度に縮小 */
}
```

### B. ui-manager.js の修正箇所

#### 1. スライダー数値更新の修正
- `SliderController.updateDisplay()` メソッドの改善
- `updateCallback` の戻り値処理の修正

#### 2. プリセット表示の改善
- アクティブ状態表示ロジックの調整

### C. main.js の修正箇所

#### 1. プリセット管理の調整
- `SimplePenPresetManager` のライブ更新処理確認
- プリセット選択時の数値表示更新

## 📂 影響ファイル

| ファイル | 修正内容 | 重要度 |
|---------|---------|-------|
| `styles.css` | プリセット配置、スライダー位置、サイズ調整 | 高 |
| `ui-manager.js` | スライダー数値更新ロジック | 中 |
| `main.js` | プリセット選択時の数値同期 | 低 |

## ⚠️ 注意事項

### 互換性の維持
- 既存の機能動作は変更しない
- イベントハンドリングは現状維持
- レスポンシブ対応は保持

### テスト項目
1. プリセット選択時の正常動作
2. スライダー操作時の数値表示更新
3. 異なるブラウザサイズでの表示確認
4. アクティブ状態の視覚的フィードバック

## 🚀 実装手順

1. **Step 1**: styles.css の修正（視覚的改善）
   - プリセット配置の均等化
   - アクティブ枠線効果の削除
   - ポップアップサイズの縮小

2. **Step 2**: スライダー数値表示の修正
   - 位置調整
   - 更新ロジックの確認

3. **Step 3**: テスト・検証
   - 各ブラウザサイズでの動作確認
   - プリセット選択の動作確認
