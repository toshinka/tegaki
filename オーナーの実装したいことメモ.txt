# オーナーの実装したいことメモ

> ブレスト用。順番は固定しない。
> 実装順は、AI が壊しにくいこと、ファイル構造上の積み上げ効率、現在フェーズとの相性を優先する。
> 達成済み・現フェーズから外れたものは、Codex が整理してよい。

---

## 現在フェーズに影響するもの

### レイヤーパネル / レイヤー編集

- レイヤーパネルは、多数レイヤー向けに今よりスリムにしたい。
- 表示情報は、目、クリップ状態、レイヤー名、透明度、サムネイルを中心にする。
- 編集操作は、レイヤー行内に詰め込むより、アクティブレイヤーに反応する操作バーや補助パネルへ集約する案がよい。
- ToonSquid の配置は参考になる。共通アイコンが上部ではなく右端に縦並びで、アクセスしやすい印象。
- 透明度は大きめのスライダーや透明度プリセットを出す方式も検討したい。ただし透明度操作の頻度はそこまで高くないかもしれない。
- レイヤーはサムネイル左右フリックで削除できるようにしたい。
- Shift 押しながらレイヤーパネルを押すと複数選択されるようにしたい。
- 複数選択したレイヤーを、一括結合、一括クリッピング、まとめて移動できるようにしたい。
- クリッピングは、同じ階層のレイヤーかフォルダに効く形がよさそう。影響を与えたいレイヤーは同階層フォルダにまとめる設計も検討する。
- レイヤーフォルダは実装済みの土台があるが、保存形式・履歴・複数操作との整合を詰めたい。
- レイヤーパネル上の `NO FRAME` 表示は、将来アニメタイムラインを 2D マトリクス化した後はクリップ情報に置き換わる可能性がある。現時点では小型化または一時非表示候補。

### UI 透明化 / ポップアップ

- 左サイドバーを半透明にし、キャンバスが重なっても裏が見えるようにして視覚圧迫を減らしたい。
- トップバーを実装したい。上下/左右反転、回転、中央リセットなどを置く。
- ステータスバーはデフォルトでは目立たせず、下側に数字中心で控えめに表示したい。
- トップバーやサイドバーは、設定や矢印アイコンでスライドして隠せるようにしたい。
- リサイズポップアップ背景を半透明にし、奥のキャンバス状態を見ながら操作できるようにしたい。最初は透明度 50% くらいで試す。
- 他のポップアップ背景やサイドバーも、最終的には透明度を統一したい。
- 各ポップアップを移動可能にしたい。クイックパネルだけ動く状態は少し違和感がある。
- 画像・アニメ出力ポップアップのプレビュー画面は、200 x 200 程度を基準にして画面からはみ出ないようにしたい。アスペクト比で多少増減してよい。

### リサイズ UI

- リサイズ画面の幅/高さ矢印は、`↔` / `↕` 選択時は均等増減、`←` / `→` / `↑` / `↓` 選択時は指定方向へ増減するようにしたい。
- 増減部分はキャンバス側にプレビュー表示したい。半透明または着色で増減範囲を示す案。
- このプレビューを見るためにも、リサイズポップアップは半透明がよさそう。

## 描画・編集ツール

- 囲い塗りを実装したい。微妙な隙間も埋められるようにしたい。
- 四角ペンを実装したい。
- ペン補正を設定できるようにしたい。最初はスライダーでよいが、最終的には筆圧カーブなどを 2D グラフ化したい。
- 設定内の項目が増えそうなので、設定 popup 内をタブで整理したい。
- エアブラシ、ぼかしツールを実装したい。
- 範囲選択、切り取り、貼り付けなどの編集ツールを実装したい。
- 直線、円、矩形ツールを実装したい。直線は Shift 押しながら引ける形も検討する。
- 定規、自在定規ツールを実装したい。
- スクリーントーンを実装したい。網点、グラデーション、縦線ストライプなど。
- 集中線、ウニフラッシュツールを実装したい。
- スポイトをサイドバーのバケツ下とクイックパネル内に置きたい。

---

## クイックパネル

- Q で出るクイックパネルを改修したい。
- カラーを多く置きたいので、より細かく整理する。
- カラーサークルとカラーセットは、`-` ボタンや `<` ボタンなどで折り畳めるようにしたい。
- ペンサイズ/消しゴムサイズのプリセットを 5 個程度置く。
- プリセットはペンと消しゴムで自動切り替えしたい。
- ツールを消しゴムに切り替えると、ペンサイズ表記から消しゴムサイズ表記へ切り替える。
- 透明度もペン透明度/消しゴム透明度で分けたい。
- スライダーと数値直接入力の両方を可能にする。
- サイズスライダーは、1 以下は 0.1 刻み、20 までは 0.5 刻み、30 以降は 5 刻み、100 以降は 10 刻み、200 以降は 50 刻みなど、低値域を広めに使える可変刻みにしたい。最大値は 500。
- プリセットを押すと別ウィンドウで詳細編集できるようにしたい。
- デフォルトに戻すボタン、現状のカラーセット保存、カラーセット呼び出しを入れたい。
- カラーセットのショートカットは `[` / `]`。
- ペンプリセットショートカットは `P + [` / `P + ]`。
- ペンサイズ初期化ボタンまたは SVG を入れたい。
- いまはペンアイコンのクリックでクイックパレットが出るが、パレット SVG をペン SVG の上に新規配置し、それをクイックパレット入口にしたい。ショートカットは Q のまま。
- ペン/消しゴム/バケツ/スポイトはショートカット使用が中心なので、サイドバーから消す判断も将来検討。ただし現時点では保留。

---

## 保存・出力・アルバム

- GIF 出力を入れる。プレビューはアニメツール確認用。
- 画像や GIF をクリップボードへコピーできるようにしたい。
- アルバムに一括選択ボタンを入れたい。
- アルバム移動する際のWINDOWSのような半透明でスムースに動かす機能は、独自実装か新ライブラリが必要そうなので保留。よいライブラリを検索できたら検討する。
- 複数選択での移動は選択の仕方による位置の入れ替え状態が煩雑になりそうなので保留。　
---

## アニメ・物理演算・長期構想

- アニメを Photoshop 的なものから、トラックやキーフレームを持つ動画ツール的なものへ変更したい。表示やデザインは ToonSquid を参考にする。
- 物理演算で XY 移動/回転させたい。Z 軸的な拡縮も考えたい。
- トラックをクリックするとグラフが出て、イージング管理できるようなイメージ。
- Live2D 的な制御も将来の参考にする。
- キャンバス外に描画物を移動して確定するとキャンバス外の絵が切れる問題は、将来的に無限キャンバスまたは外部領域保持で解消したい。物理演算アニメ化では必須。

---

## 入力・端末操作

- タブレットやスマホ環境はオーナー側で確認できないため、ピンチ操作などのハンドジェスチャー対応は慎重に進める。

---

## 完了済み・現フェーズでは保留したもの

- ペンの遅いストロークが丸の集合に見える問題は Phase 1e で改善済み。
- 筆圧入力、筆圧 ON/OFF ショートカットは Phase 1e で実装済み。
- V パネルの水平/垂直反転、`H` / `Shift+H`、回転 360 度化、数値ダブルクリック入力は Phase 1i で実装済み。
- レイヤー透明度の数値ドラッグは Phase 1i で実装済み。
- クリップボード画像貼り付け、アクティブレイヤーコピー/新規コピーレイヤー作成は Phase 1i で土台実装済み。
- PNG エクスポート、プロジェクト保存/読み込み、アルバムHTML保存/読み込みは Phase 1g〜1m で復旧済み。
- アルバムのドラッグ並べ替え、HTMLエクスポート/インポート、一括削除/一括書き出しの土台は Phase 1l〜1m で実装済み。
- レイヤー削除ボタンや popup 閉じるボタンの共通化は Phase 1k で実装済み。

## 追加メモ
・　Vキーによるアクティブレイヤー編集の回転で、楕円軌道設定もさせたい。縦と横と確度のパラメーターを数字とスライダーで調整？アイコンを押すとパネルが拡張して設定できるみたいな。
・　漫画のコマで使える自動マスキングみたいなの。□の外側はマスキングされるみたいな。図形を変形させると線に合わせてマスキングの形も追従して変わる。アニメツール的には目や口の開閉に応用できそうな？特別な判定の線にひっつく特別なマスク塗料みたいなのが良いのだろうか？特別な線（大丈夫なら通常線でもいいけど）で四角を描いて外側にマスク塗料でバケツ塗り。その死角を変形させると塗装部分も動く。四角の中でも良い。線の上に太めの線でマスク塗料で塗ると追従する太い線になるみたいな？なんか良い案は無いか。マスクに限らずそういう判定になるクリッピングレイヤーとか？
・　アクティブレイヤー編集パネルに、初期化リセットボタンを置く。色々と動かした後にパネル開く前の状態に戻すのは大変なので。
・　ペンや色のパネルを司るクイックパネルだけど、後に実装予定の選択ツールをアクティブにすると、パネルがフリップして矩形・円・折れ線とか、メッシュ変形とかのパネルに変わるというのはどうだろうか？選択ツール中はペンで絵を描けないのでパネル資産を有効に使う。多分他に無さそうなアイデア？ツールによってクイックパネルが入れ替わることで、サイドバーにずらりとアイコンを並べる必要が無くなる。ペン・消しゴム・バケツもクイックパネル内のみにするかは悩みどころ。現状はステータス見れば分かるからサイドバーで知らせなくても良いという考え。Tool:✑ペンみたいにして、アイコンも入れると文字だけよりも視覚処理しやすくなり、サイドバーに入れる必要が無くなるだろうか？要検討。
・　クイックパネル内アイコンは、ペン/消しゴム/バケツ/エアブラシ/図形/選択の予定。スポイトはカラーサークル周りに置く。
・　ショートカットでクイックパネルを展開できないペンだけでなんとかしたい人向けで、設定でクイックパネルを開閉する糊代を設けて、小さくたためるようにする。✘ボタンみたいにパネルを消すのではなく小型化するみたいな。この設定にした場合は✘ボタンは消えるとか、そこに小型化アイコンを置くとか？ペン先で自在に動かすための箇所と展開のON/OFFをする判定箇所を設けながら、触りやすさと小ささのバランスを取る。
・　数字入力系は数値入力用ポップアップが出現できるようにするとか？煩雑かな？今のところ優先度はかなり低め。
・　設定内で、項目ごとにタブを切れ替えるというアイデアを前述したかもだが、画像の出力の中にも入力のタブを入れたい。アイコンが写真に↓マークのSVGを使ってるが、矢印の無いものか、上下矢印のあるものにする。出力はそのままで入力タブに切り替えると、ファイル選択とクリップボードからの貼り付けを選べる。前にCTRL+Vでクリップボード貼り付けとしてたが、こっちのが制御しやすいと思われる？現在のキャンバスに合わせてアスペクトを保持して縮小するかのチェックボタンもいれる。

## 追加メモ2
・　ペンサイズ等の設定は環境設定の中の保存ボタン等でhtmlに記憶される。アルバムのjsonにも保存時に記録され、ロードやインポート時に反映される。
・　アクティブフォルダでVキーでのレイヤー編集モードを使うと、フォルダの中の全てを変形・移動させられる。今後実装予定の選択範囲での変形や移動、物理演算アニメでも同様。絵は描けないがフォルダ内のまとめての変形はできる。
・　Shift+DELでアクティブレイヤー/フォルダ削除。全削除は設定の中にボタンを入れる。
・　レイヤーパネルと右サイドバーが、縦スライダーが入るとは言え離れすぎてる。枠線が消えない良い位置にする。
・　クイックパネルは横幅が今の60％に近づけるのを目安にスリム化する。アイコンも小型化したり詰めたりする。
・　優先度はそこまで高くないが文字入力モードを。漫画のセリフとかでよく使われてるフリーフォントを導入できるならしておきたい。
・　アルバムは触ったのがいきなりロードされるとビックリしたり、下記途中の絵が…みたいになるのでクリックは選択のみにする。つまり最初から選択モードであるので、選択ボタンは不要になる。一つだけの選択の時のみ、ロード？ボタンが押せる。現在の状態を保存も含めてSVGからテキトーなの拾ってアイコン化でも良いかも。
