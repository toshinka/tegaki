# Phase 4A11 問題点と解決案

## 🔍 特定された問題点

### 1. GUI表示の問題
- **ステータスウィンドウが右にめり込む**: dat.GUIが右サイドバーに配置されているが、レイアウトが崩れている

### 2. 座標変換の問題
- **レイヤー移動時に画像が外に吹き飛ぶ**: transform-utils.jsの実装が不正確
- **バケツツールで座標がズレる**: WebGL描画とCanvas2D座標系の不一致
- **Ctrl+Zで画像の不一致**: モデルマトリックスの保存・復元が正しくない

### 3. 座標系の統一問題
- **WebGL座標系とCanvas2D座標系の混在**: Y軸反転処理が複数箇所で発生

### 2. transform-utils.js の実装問題

現在のコードでは `transform-utils.js` が正しく実装されていない可能性があります。

```javascript
// transform-utils.js の正しい実装例
export function reset(matrix) {
    glMatrix.mat4.identity(matrix);
}

export function translate(matrix, x, y) {
    glMatrix.mat4.translate(matrix, matrix, [x, y, 0]);
}

export function rotate(matrix, radians) {
    glMatrix.mat4.rotateZ(matrix, matrix, radians);
}

export function scale(matrix, sx, sy) {
    glMatrix.mat4.scale(matrix, matrix, [sx, sy, 1]);
}

export function getTranslation(matrix) {
    return {
        x: matrix[12],
        y: matrix[13]
    };
}

// 重要: ワールド座標→ローカル座標変換
export function transformWorldToLocal(worldX, worldY, modelMatrix) {
    const invMatrix = glMatrix.mat4.create();
    glMatrix.mat4.invert(invMatrix, modelMatrix);
    
    const worldPos = glMatrix.vec4.fromValues(worldX, worldY, 0, 1);
    const localPos = glMatrix.vec4.create();
    glMatrix.vec4.transformMat4(localPos, worldPos, invMatrix);
    
    return {
        x: localPos[0],
        y: localPos[1]
    };
}
```

### 3. レイヤー移動の修正

```javascript
// core-engine.js の onPointerMove 内のレイヤー移動処理を修正
onPointerMove(e) {
    if (this.isLayerTransforming) {
        if (!this.transformTargetLayer) return;

        // スクリーン座標の差分を計算
        const dx = (e.clientX - this.transformStartX) / this.viewTransform.scale;
        const dy = (e.clientY - this.transformStartY) / this.viewTransform.scale;
        
        // 元のマトリックスをクローン
        const newMatrix = glMatrix.mat4.clone(this.originalModelMatrix);

        if (this.transformMode === 'move') {
            // 単純な平行移動を適用
            translate(newMatrix, dx, dy);
        } else if (this.transformMode === 'rotate_scale') {
            // 中心点を基準とした回転・スケール
            const centerX = this.width / 2;
            const centerY = this.height / 2;
            
            // 1. 中心に移動
            translate(newMatrix, centerX, centerY);
            
            // 2. 回転とスケールを適用
            const angle = dx * 0.01; // 回転量を調整
            const scaleFactor = Math.max(0.1, 1 + dy * 0.001); // スケール量を調整
            rotate(newMatrix, angle);
            scale(newMatrix, scaleFactor, scaleFactor);
            
            // 3. 中心から戻す
            translate(newMatrix, -centerX, -centerY);
        }

        this.transformTargetLayer.modelMatrix = newMatrix;
        this.renderAllLayers();
        return;
    }
    // 残りのコードは同じ...
}
```

### 5. 履歴保存・復元の修正

```javascript
// core-engine.js の saveState と restoreState を修正
saveState() {
    if(this.isLayerTransforming) return;
    const state = {
        layers: this.app.layerManager.layers.map(layer => ({
            name: layer.name,
            visible: layer.visible,
            opacity: layer.opacity,
            blendMode: layer.blendMode,
            imageData: new ImageData(
                new Uint8ClampedArray(layer.imageData.data),
                layer.imageData.width,
                layer.imageData.height
            ),
            // Float32Array として保存
            modelMatrix: Array.from(layer.modelMatrix)
        })),
        activeLayerIndex: this.app.layerManager.activeLayerIndex
    };
    this.history = this.history.slice(0, this.historyIndex + 1);
    this.history.push(state);
    this.historyIndex++;
}

restoreState(state) {
    this.app.layerManager.layers = state.layers.map(layerData => {
        const layer = new Layer(layerData.name, layerData.imageData.width, layerData.imageData.height);
        layer.visible = layerData.visible;
        layer.opacity = layerData.opacity ?? 100;
        layer.blendMode = layerData.blendMode ?? 'normal';
        layer.imageData.data.set(layerData.imageData.data);
        
        // modelMatrix の復元を確実に行う
        if (layerData.modelMatrix) {
            if (Array.isArray(layerData.modelMatrix)) {
                layer.modelMatrix = glMatrix.mat4.fromValues(...layerData.modelMatrix);
            } else {
                layer.modelMatrix.set(layerData.modelMatrix);
            }
        } else {
            // デフォルト値を設定
            glMatrix.mat4.identity(layer.modelMatrix);
        }
        
        layer.gpuDirty = true;
        return layer;
    });
    this.app.layerManager.switchLayer(state.activeLayerIndex);
    this.renderAllLayers();
}
```
