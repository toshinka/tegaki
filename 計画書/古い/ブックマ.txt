【大規模ファイルの分割出力ルール】

■ 適用条件
・ファイルが1500行を超える場合、複数パートに分割して出力する
・各パートは800-1000行を目安とする

■ 出力フォーマット
```javascript
// ================================================================
// [PART X/N] セクション名
// ================================================================
// ⚠️ このファイルはNパートに分割されています
// ⚠️ 各パートを順番にコピペして結合してください
// ================================================================

// コード本体

// ================================================================
// [END PART X] - 次は PART X+1 をこの下に貼り付けてください
// ================================================================
```

■ 分割基準
1. クラス定義単位で分割（クラスの途中で切らない）
2. 関連する関数群をまとめる
3. PART 1 に集約すべきものは集約する。

■ 注意事項
・各パート出力時に「PART X/N を出力しました。続きを出力しますか？」と確認
・パート間で変数名・関数名の整合性を保つ
・最終パートでファイル全体の概要を説明
