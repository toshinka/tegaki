- **サムネイル生成**: PixiJS RenderTexture活用

```javascript
// 非破壊変形の実装例
transform(layerId, transformMatrix) {
    const layer = this.layers.get(layerId);
    
    // 元データ保存（初回のみ）
    if (!layer.originalData) {
        layer.originalData = cloneDeep(layer.vectorStrokes);
    }
    
    // 新Container生成による非破壊変形
    const newLayer = this.createTransformedLayer(layer, transformMatrix);
    return newLayer; // 元データ保持
}
```

zoomTo(zoomLevel, centerPoint, animated = true) {
    if (animated) {
        this.animateZoom(zoomLevel, centerPoint);
    } else {
        this.applyZoom(zoomLevel, centerPoint);
    }
}

## 📊 統合連携の主要イベントフロー

### 描画フロー
```
1. UI操作 (PixiUIController)
   ↓ tool:change イベント
2. ツール切り替え (PixiToolProcessor)
   ↓ draw:start/continue/end イベント
3. 描画処理 (PixiInputController)
   ↓ tool:drawing:ended イベント
4. レイヤー更新 (PixiLayerProcessor)
   ↓ layer:strokeAdded イベント
5. UI反映 (PixiUIController)
```

### カラー変更フロー
```
1. カラー選択 (PixiUIController)
   ↓ color:select イベント
2. カラー処理 (ColorProcessor)
   ↓ color:changed イベント
3. ツール設定更新 (PixiToolProcessor)
```

### キャンバス操作フロー
```
1. キャンバス操作 (PixiUIController)
   ↓ canvas:zoom/pan イベント
2. ビューポート更新 (CanvasController)
   ↓ canvas:zoomed/panned イベント
3. UI調整 (PixiUIController)
```
