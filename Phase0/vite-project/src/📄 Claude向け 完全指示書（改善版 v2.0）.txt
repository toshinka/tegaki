## 🏗️ SOLID原則 完全準拠要件

### S: 単一責任原則（Single Responsibility）
- **責務**: Perfect Freehandを使用したベクターペンの描画処理のみ
- **禁止**: レイヤー管理、履歴管理、ファイル保存等の他責務混入

### O: 開放閉鎖原則（Open-Closed）
- **拡張**: 新しいベクターツールはこのクラスを参考に独立実装
- **修正**: 既存コードの修正なしで新機能追加可能な設計

### L: 置換可能性（Liskov Substitution）
- **インターフェース**: 同種ツールと完全置換可能なメソッド群
- **戻り値**: 一貫した型（getStroke() → Perfect Freehand形式）

### I: インターフェース分離（Interface Segregation）
- **必要機能のみ**: ブラシやテキスト等、無関係機能の実装強制なし
- **メソッド**: ペン描画に必要な最小限のメソッドのみ

### D: 依存性逆転（Dependency Inversion）
- **抽象依存**: Perfect Freehandライブラリ（抽象）に依存
- **具体依存禁止**: 他の具体クラスへの直接new禁止

## 💡 実装ヒント

import { getStroke } from 'perfect-freehand'

// 点の収集
this.points.push([x, y, pressure])

// ストローク生成
const stroke = getStroke(this.points, this.options)

// WebGL用頂点データ変換
const vertices = stroke.flatMap(point => [point[0], point[1]])

// 筆圧を線幅に反映
const effectiveWidth = this.width * (0.5 + pressure * 0.5)

// Perfect Freehandオプションに反映
const options = {
  ...this.options,
  size: effectiveWidth
