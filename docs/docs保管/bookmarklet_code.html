📝 Tegaki めぶきちゃんねる連携

めぶきちゃんねるで直接お絵かきできるブックマークレットです。
描いた絵をそのまま投稿できます。

🔖 ブックマークレットのインストール

手順:
  1. 下の緑色のボタンをブックマークバーにドラッグ&ドロップ
  2. または右クリック → 「お気に入りに追加」
🎨 Tegaki起動

↑ このボタンをブックマークバーにドラッグ

📖 使い方

  1. めぶきちゃんねるのスレッドを開く
  2. ブックマークバーの「🎨 Tegaki起動」をクリック
  3. お絵かきツールが全画面表示される
  4. 絵を描く
  5. 「✓ 掲示板に貼り付けて閉じる」をクリック
  6. 自動的に投稿フォームに画像が添付される
  7. コメントを入力して投稿

✨ 機能

  • ✓ ベクターペン描画
  • ✓ レイヤー機能
  • ✓ アニメーション対応 (GIF/APNG/WebP)
  • ✓ Undo/Redo対応
  • ✓ ズーム・パン操作
  • ✓ 出力フォーマット選択可能

⌨️ ショートカット

  • P - ペンツール
  • E - 消しゴム
  • Space + ドラッグ - キャンバス移動
  • Ctrl + ホイール - ズーム
  • Ctrl + Z - 元に戻す
  • Ctrl + Y - やり直し
  • Alt + A - タイムライン表示/非表示
  • Alt + C - 新しいカット追加

⚙️ 技術仕様

対応掲示板:
  • めぶきちゃんねる (mebuki.moe) ✓
  • その他の掲示板は今後対応予定
出力フォーマット:
  • GIF - アニメーション対応、広い互換性
  • PNG - 静止画、高画質
  • APNG - アニメーション、PNG互換
  • WebP - 軽量、最新ブラウザ対応

🔧 開発者向け情報

ローダースクリプトのURL:

javascript:(function(){if(!window.tegakiStart){var s=document.createElement('script');s.charset='UTF-8';s.src='https://toshinka.github.io/tegaki/v8.13_History28/tegaki-loader.js';document.body.appendChild(s)}else{window.tegakiStart()}})();

📝 注意事項

  • ベクターペン主体のため、複雑な絵はデータ量が増えます
  • アニメーションのフレーム数が多いと出力に時間がかかります
  • ブックマークレット起動中は他の操作ができません
  • 描画内容はキャンセルすると破棄されます（保存機能なし）

🐛 トラブルシューティング

ブックマークレットが動かない場合:
  • ブラウザのコンソール（F12）でエラーを確認
  • めぶきちゃんねるのDOM構造が変わった可能性
  • ブラウザのキャッシュをクリアして再試行
画像が投稿フォームに反映されない場合:
  • 手動で「レスを投稿」ボタンを押してフォームを開く
  • ファイル入力欄が表示されているか確認
  • ブラウザのFile API対応状況を確認
