## ⚠️ Claude実装時の重要注意点

### 必須チェック項目
```typescript
// 1. インポート文の正確性
import { EventBus } from '../core/EventBus.js';  // .js拡張子必須
import * as PIXI from 'pixi.js';  // PixiJS統一インポート

// 2. ふたば色の正確な数値
const FUTABA_MAROON = 0x800000;  // 16進数、#付けない
const FUTABA_BACKGROUND = 0xffffee;

// 3. サイズ値の厳密遵守
width: 56px;  // ツールボタン
width: 32px;  // 色スウォッチ  
width: 80px;  // ツールバー

// 4. 型安全性の確保
interface IEventData {  // any型禁止
  'drawing:start': { point: PIXI.Point; pressure: number };
}

// 5. エラー処理の必須実装
try {
  await this.pixiApp.init(config);
} catch (error) {
  console.error('初期化失敗:', error);
  return false;
}
```

### ファイル作成順序の厳守理由
1. **型定義先行** → 他ファイルでimport依存
2. **EventBus早期** → 全システム通信基盤  
3. **PixiApplication基盤** → 描画エンジンの前提
4. **入力→描画→ツール** → データフロー順序
5. **UI最後** → 全機能統合後のインターフェース
